# 企画告知用フォーマット（広告用）

## フォーマット概要
商品やサービスの企画を告知する際の広告用フォーマットです。興味を引き、行動を促す構成になっています。

## 構造
```
[企画の告知] - 何の企画かを明確に
[ベネフィット] - 参加者にとってのメリット
[詳細情報] - 具体的な内容や条件
[行動喚起] - 参加方法や次のステップ
[ハッシュタグ] - 関連するハッシュタグ
```

## 使用例
```
🚀 新企画のお知らせ！

「Laravel実践講座」を開催します

✅ 実務で使える技術を3時間で習得
✅ 実際のプロジェクトを題材にした実習
✅ 質問し放題のサポート付き

日時: 2月15日(土) 14:00-17:00
場所: オンライン（Zoom）
参加費: 5,000円

詳細・お申し込みはDMまで！ #Laravel #講座 #プログラミング #実践
```

## 効果的な要素
- 明確な企画の告知
- 具体的なベネフィット
- 限定感や緊急性の演出
- 明確な行動指示
