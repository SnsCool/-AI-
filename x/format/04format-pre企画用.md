# 企画告知用フォーマット（一般用）

## フォーマット概要
一般的な企画やイベントを告知する際のフォーマットです。親しみやすく、参加しやすい印象を与えます。

## 構造
```
[企画の紹介] - 何の企画かを親しみやすく
[内容の説明] - 企画の詳細や魅力
[参加方法] - どう参加するか
[締切・注意事項] - 重要な情報
[ハッシュタグ] - 関連するハッシュタグ
```

## 使用例
```
📢 お知らせ

来週、技術勉強会を開催します！

テーマ: 「最新のWeb開発トレンド」
- フロントエンドの最新技術
- バックエンドの設計パターン
- 実際のプロジェクト事例

日時: 2月20日(木) 19:00-21:00
場所: オンライン
参加費: 無料

興味のある方は、リプライで参加表明をお願いします！ #勉強会 #Web開発 #技術 #無料
```

## 効果的な要素
- 親しみやすい表現
- 具体的な内容の説明
- 参加しやすい条件
- 簡単な参加方法
