# ストーリー型投稿フォーマット

## フォーマット概要
個人的な体験談を通じて信頼性と親近感を演出し、読者の共感を得るフォーマットです。

## 構造
```
[体験の紹介] - 個人的な体験や出来事を紹介
[詳細な経緯] - どのような過程でその結果に至ったか
[学びや気づき] - その体験から得た教訓
[読者への問いかけ] - 読者の経験や意見を聞く
```

## 使用例
```
実は私、3ヶ月で年収2倍になった話

フリーランスエンジニアとして独立したばかりの頃、月収15万円で生活に困っていました。

そこで、AIツールを活用した効率化と、単価の高い案件への挑戦を始めました。

結果、3ヶ月で月収30万円を達成。

あなたも同じような経験ありますか？
```

## 効果的な要素
- 個人的で具体的な体験談
- 具体的な数値や期間
- 学びや気づきの共有
- 読者との対話を促す問いかけ
