# 共感型投稿フォーマット

## フォーマット概要
読者の自己認識と共感を促す投稿フォーマットです。多くの人が「自分もそうだ」と思える内容で拡散力を高めます。

## 構造
```
[対象者の特定] - 特定の職業や立場の人を指す
[共通の特徴] - その人たちに共通する特徴や行動
[共感を促す要素] - 読者が「自分もそう」と思える内容
[追加の洞察] - より深い共感や理解を促す内容
```

## 使用例
```
プログラマーな人、だいたいコーヒー中毒

朝はブラックコーヒー、午後はエナジードリンク、夜はまたコーヒー。

コードが動かないときの唯一の味方。

あなたもそうでしょ？
```

## 効果的な要素
- 特定の職業や立場を明確に指定
- 多くの人が経験する共通の行動や特徴
- 親しみやすい表現
- 読者への問いかけでエンゲージメント促進
