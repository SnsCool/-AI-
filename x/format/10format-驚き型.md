# 驚き型投稿フォーマット

## フォーマット概要
読者の好奇心を刺激し、意外性のある情報でクリックを促すフォーマットです。

## 構造
```
[驚きの表現] - 読者の予想を裏切る導入
[意外な事実] - 多くの人が知らない情報
[詳細の説明] - なぜそうなのかの理由
[行動の促し] - 読者に何かをしてもらう呼びかけ
```

## 使用例
```
え、知らなかった…

ChatGPTは実は無料で使える

有料版のGPT-4を使わなくても、GPT-3.5なら無料で十分使えます。

月額20ドル払う前に、まずは無料版で試してみてください。

意外と高機能なんです。
```

## 効果的な要素
- 読者の予想を裏切る意外性
- 多くの人が知らない情報
- 具体的で実用的な内容
- 読者の行動を促す呼びかけ
