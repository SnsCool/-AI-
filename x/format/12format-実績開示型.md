# 実績開示型投稿フォーマット

## フォーマット概要
具体的な数値や実績を提示することで説得力を高め、読者の信頼を得るフォーマットです。

## 構造
```
[行動の説明] - 何をしたかを具体的に説明
[結果の提示] - 具体的な数値や成果
[期間の明記] - どのくらいの期間で達成したか
[方法の共有] - どのような方法で達成したか
```

## 使用例
```
毎日1時間勉強したら3ヶ月でTOEIC800点になった

元々のスコアは450点でした。

毎日1時間、オンライン英会話と単語帳を継続。

3ヶ月後、TOEIC800点を達成。

継続は力なり、を実感しました。
```

## 効果的な要素
- 具体的な数値や期間
- 明確な行動と結果の関係
- 再現可能な方法
- 読者のモチベーションを高める内容
