# 基本投稿フォーマット

## フォーマット概要
最も基本的なX投稿フォーマットです。シンプルで効果的な投稿を作成するためのテンプレートです。

## 構造
```
[フック] - 読者の注意を引く最初の一文
[本文] - メインの内容（2-3文）
[問いかけ] - 読者とのインタラクションを促す
[ハッシュタグ] - 関連するハッシュタグ（1-3個）
```

## 使用例
```
なぜ多くの開発者がテストを書かないのか知っていますか？

実は、テストの重要性を理解していても、書き方が分からないからです。

今日は、初心者でも書けるテストの書き方を3ステップで説明します。

あなたはテストを書いていますか？ #プログラミング #テスト #初心者
```

## 効果的な要素
- フックは疑問文または驚きの事実
- 本文は簡潔で分かりやすく
- 問いかけでエンゲージメントを促進
- ハッシュタグは適度に使用
