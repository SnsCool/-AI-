# 引用RT基本フォーマット

## フォーマット概要
他の投稿を引用してリツイートする際の基本フォーマットです。元の投稿に価値を追加します。

## 構造
```
[引用元の要約] - 引用する投稿の要点
[自分の意見] - 追加の洞察やコメント
[関連する質問] - 読者との議論を促す
[ハッシュタグ] - 関連するハッシュタグ
```

## 使用例
```
「テストを書く時間がない」という意見について

確かに時間はかかりますが、長期的には開発速度が向上します。

実際、テストがあることで安心してリファクタリングでき、結果的に開発効率が上がりました。

あなたはテストの時間をどう確保していますか？ #テスト #開発効率 #リファクタリング
```

## 効果的な要素
- 引用元の内容を正確に要約
- 自分の経験や知識を追加
- 建設的な議論を促す
- 専門性を示す内容
