  会議概要
  決定事項
  1. 報酬設計（確定）
  方向A: 知足側→X顧問への送客
  フロー: 興味ある人→面談→VIP提案→不成約ならXコモンへ
  方向B: X顧問→知足への送客
  知足入会1件あたり 12万円（10%）をLevelaに支払い
  ゆーへー「ナギたちにちょっと多めに返せるようにできたらめっちゃいいな」
  2. コミュニティ構造（確認）
  IAとDIGは統合（1コミュニティとして扱う）
  知足側: 知足 / IA+DIG / 後日リスト / ゆーへーX発信 / 知足説明会リスト400名（NEW）
  Levela側: VIP会員（現5名） / Xコモンアクティブ200-300名 / 累計1,000名（休眠700名） / SnsClub
  3. セミナー形式（確定）
  「月次報告会」ライブ配信
  ゆーへー × ロジン × 凪の3名で毎月1回
  構成:
    先月の売上・成果・失敗談を生々しく報告
    X/SNS/AIのトレンド振り返り
    来月の動き予告
  金額はピー音で伏せる → 聞きに来ないと聞けない
  アーカイブは一絲側で動画編集 → 各チャネル配布 + YouTube
  「番組」としてファンがつく形を目指す
  知足の生々しい数字も出す（入会者数・成約件数・売上等）
  koujitsuのクライアント事例や資料も出せる
  たまにお客さんにゲスト出演してもらうのもあり
  Zoom配信 + 同時配信も検討
  4. Levelaから知足への送客チャネル（具体）
  VIP会員（現5名）に「知足もやった方がいいよ」とシンプルに案内
  入会者全員の面談時に知足を提案
  Xコモンアクティブ200-300名に知足を発信
  累計1,000名（休眠700名含む）にもアプローチ
    「X以外にも事業戦略やらないとな」という層が絶対いる
  SnsClub卒業生にも案内（インスタ伸びたけど経営学びたい層）
  5. 知足側からX顧問への送客チャネル
  知足コミュニティのLINE/チャットでX顧問を案内
  ゆーへーX発信（月間3,000万imp）で「Levelaと連携してる」と発信 → 一番取れそうとの判断
  セミナー時にLevela/ロジンを紹介
  知足説明会リスト400名（全員経営者）にもセミナー案内
  リアルバリューとのコラボセミナーも可能
  6. Levelaの出し方
  ゆーへーさんの横並びのポジション（マーケor事業戦略）
  「知足とLevelaが組んでるんだ」という事実自体がブランディング
  ロジンとゆーへーのX相互発信戦略を先に決める
  ロジンからゆーへーのXにインプを飛ばす→そこからも流入
  7. LP制作の連携
  koujitsu側からLP制作の問い合わせが来ている
  Levelaは広告チームがLP量産体制あり
  何本/月、デザインの型、金額感を共有してくれれば対応可能
  8. 契約・請求
  VIPプラン契約書: クラウドサインで送付
  請求書: 株式会社koujitsu 担当:柴田 宛（ゆーへーの個人メールに送付）
  支払い: 4月末（経理連絡済み）
  koujitsu払い（一絲ではなく）→ コンサル案件も取れる可能性があるため
  9. プレスリリース
  知足×Levela提携のプレスリリースを出す
  イベントページも同時作成
  リリースと同時に全員でX拡散（凪・ゆーへー・加藤・ロジン・コマ）
  4月・5月のセミナー日程もセットで告知
  10. 知足の説明資料
  Levelaがアプローチする際に使える簡易説明資料が必要
  加藤さんが準備
  知足の最新実績（会議中に共有）
  2-3月に過去最高売上報告が10社以上
  リフォーム山本くん: 月半億3,000万（過去最高）
  別の受講生: 売上4,000万増
