📝 セッション議事録：目標達成に向けた戦略の再構築
今回のセッションでは、やりたいことの増加による優先順位の整理と、卒業時のゴール「月収100万円」を確実に達成するための逆算型KPI設計を行いました。
1. 現在の状況と心境
• 
やりたいことの膨張: インスタ運用、マネタイズ合宿への参加、スクラブ（スクール）講師への挑戦など、活動の軸が多角化している状態 。

• 
講師への想い: 「誰かに教え、その成長の対価をいただくこと」に強いやりがいを感じており、本職（管理職）の経験も活かしたいという強い意志がある 。

• 
目標設定の再確認: 複数の軸がある中で、現在の目標設定やKPIが正しい道に進んでいるかを整理したい 。

2. 目指すべき「最高到達地点」の言語化
キヨさんが「スター状態（絶好調）」で突き進むための各軸のゴールを定義しました。
• スクラブ講師の軸:
    ◦ 
事実: 歴史上類を見ない「クラス全員が満アカ（1万人）」を達成し、スペインマーカーを多数輩出する 。

    ◦ 
評価: 生徒からの満足度評価で10点満点中「10点」を全員から獲得する 。

• マネタイズ（アカウント運用）の軸:
    ◦ 
事実: スクール卒業までに月収100万円を達成する 。

    ◦ 
意図: プラットフォームに依存しない「自らの能力で市場を理解し、モノを売る力」を証明する 。

3. 5月のゴール（100万円）からの逆算設計

4. 直近（3月第1週〜）のアクションプラン
迷いを無くし、スピードを最大化するための具体行動です。
• 
コンセプトの再定義: 週末までにマネタイズに特化したアカウントコンセプト・プロフィールを完全に決めきる 。

• 
成功者へのコンタクト: 既に成果を出している先人（ミキさん等）10名にDMを送り、ヒアリングのアポイントを取る 。

• 質の高いインプット:
    ◦ グループコンサルへの参加（3月中に計5回） 。

    ◦ 過去アーカイブの視聴（週4本ペース） 。

•
