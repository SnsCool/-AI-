議題：メディア部のシズルへの移管 / AOさんYouTube / マニュアル作成 / 定例会議 / AI活用
■ 議題1：メディア部のシズル（しずれ）への運用移管
【背景】
レベラポータルの「巻き取り」ページでメディア部を管理。対象：レベラYouTube（密着）、AOさんYouTube、Instagram等
【決定事項】
シズルが実動部隊としてメディア部を全権巻き取り。単価7,000円、3月から運用開始
既存ディレクター・運用担当者の収入が下がらないよう調整（運用担当者は3,000円単価で問題なし、ディレクターはシズル側で計算し収入維持を前提に引き継ぎ）
【引き継ぎ方針】
既存メンバーをシズルの体制内に組み込む。YouTube密着はマニュアル化済み、Instagramは既存条件との調整が必要
引き継ぎ期間は定例会議での意思疎通が重要。橋本さんとの連携で進める
【課題】
ローンチ（新規アカウント立ち上げ）が属人化 → 当面停止し、売れるアカウント作りを優先
SNS解説系は伸びるパターンあり。エバーグリーン配布で回し、ストーリー教育を属人化しない体制が必要（スケール課題）
■ 議題2：AOさんのYouTube運営
【現状】
8本撮影済み（会話形式）。月20時間稼働（撮影のみ）。企画はAOさんが決定（過去事例＋未実施ネタから選定）
編集は石原彩音さん（外部委託）。KPI：1投稿4,000再生で半年回収。スクラブ誘導は最後にチラッと程度
【課題・改善点】
YouTube知見が社内に不足 → 01（ゼロイチ）で進める必要。競合分析が未実施 → 徹底リサーチ必要
動線設計（スクラブ誘導）の改善が必要。AOさん過去データ分析（文字起こし→伸びた動画→競合パターンをAO流に適用）
【決定事項】
2/27にAOさん打ち合わせ設定（方向性の認識合わせ）。事前に競合分析を実施して持ち込む
ゆなも打ち合わせに参加。Discordの「青YouTube」チャンネルで連絡
■ 議題3：マニュアル作成
YouTube密着の業務ワークフロー（企画出し→ヒアリング→構成→撮影→編集→確認）をドキュメント化
ゆなが2月中にマニュアル作成。完成後、頼藤さんに共有しフィードバック
毎週火曜日の定例で不明点があればコメントを入れる
■ 議題4：定例会議・報告体制
週次定例：火曜14時（小牧・足立・安藤・ゆな＋金森）。30分〜1時間
月次：新規事業ミーティング（毎月4日 10時）。金森さん必須出席。幹部会議でメディア部報告
報告テンプレート：目標→先週実施→未完了→今週予定（フォーマット統一）
■ 議題5：運用代行の利益率
会社平均利益率は約40%、平均給与40万円。作業者コストが圧迫している構造
利益率30%でも問題なし（人件費中心）。新規月10件増加ペース。既存顧客へのアップセル余地あり
■ 議題6：Claude Code / AI活用（安藤より紹介）
ツール紹介：Warp（ターミナル）/ Claude Code（Opus 4.6）/ クリッピー（クリップボード管理）/ CLAUDE.md / クロードメム（記憶システム）/ GLM-Z.ai（安価な外部AI）/ スキルズ（技マシン）
活用方針：YouTube分析にClaude Code活用、AOさんチャンネル文字起こし→分析、競合リサーチ自動化、ルーティン業務のスキルズ化を推進
■ 決定事項まとめ
① メディア部をシズルが3月から巻き取り（単価7,000円・全権委任）
② 既存ディレクターの収入維持を前提に引き継ぎ
③ 2/27にAOさんYouTube打ち合わせ（安藤・ゆな・金森参加）
④ ゆなが2月中にYouTubeマニュアル作成
⑤ 金森さんは火曜定例（週次）と月次会議に参加
⑥ ローンチ一旦停止、売れるアカウント作りを優先
⑦ エバーグリーン配布で属人化しない体制を構築
⑧ Claude Code / スキルズの活用を推進
