## リール公開添削会

# Instagramチェックリスト100

### 第1章：戦略・企画編 (1-15)

[ ] 目的の明確化: このリールで何を達成したいか？ (認知拡大、ファン化、商品購入など)

[ ] ターゲット設定: 誰に届けたいリールか？ (年齢、性別、興味関心などペルソナは明確か)

[ ] アカウントコンセプトとの一貫性: アカウント全体のテーマや世界観と、リールの内容は一致しているか？

[ ] プロフィールとの連動: プロフィールの自己紹介文とリールの内容に矛盾はないか？

[ ] 提供価値の定義: 視聴者はこのリールを見ることで何を得られるか？ (知識、共感、娯楽など)

[ ] 競合分析: 同じジャンルの人気リールの構成やトレンドを分析・参考にしているか？

[ ] 独自性の確保: 他のアカウントにはない、自分ならではの切り口や強みは何か？

[ ] テーマの絞り込み: １つのリールで伝えたいメッセージは１つに絞られているか？

[ ] 視聴者目線の徹底: 自分の「伝えたいこと」だけでなく、視聴者が「知りたいこと」になっているか？

[ ] ゴール設定: 視聴後にどのような行動を促したいか明確か？ (いいね、コメント、保存、フォローなど)

[ ] シリーズ化の検討: 今回のリールは、他の投稿と関連性を持たせられるか？

[ ] サムネイルの構想: 一覧で表示された時に、クリックしたくなるサムネイルをイメージできているか？

[ ] キーワードの選定: ターゲットが検索しそうなキーワードを意識しているか？

[ ] 投稿タイミングの戦略: ターゲット層が最もアクティブな時間帯を把握しているか？

[ ] コンプライアンスの確認: 表現や内容に、誤解を招くものや不適切な点はないか？

### 第2章：構成・シナリオ編 (16-30)

[ ] 冒頭3秒のインパクト: 最初の3秒で、視聴者の興味を引く映像や言葉を入れているか？ (結論、ハイライト、意外性など)

[ ] 結論ファースト: 動画の冒頭で「何についての動画か」が明確に伝わるか？

[ ] 離脱ポイントの排除: 自己紹介や無関係なシーンなど、視聴者が飽きてしまう冗長な部分はないか？

[ ] テンポの良い展開: シナリオ全体が、視聴者を飽きさせないテンポで進むか？

[ ] 情報の順序: 視聴者が最も理解しやすい順番で情報が提示されているか？

[ ] 共感・問いかけの活用: 「〜なことない？」のように、視聴者が「自分ごと」として捉えられる問いかけがあるか？

[ ] ストーリー性: 単なる情報の羅列ではなく、視聴者が感情移入できるストーリーがあるか？

[ ] 有益性の担保: 初心者や知識がない人でも理解できるよう、丁寧に解説されているか？

[ ] 具体例の活用: 視聴者の理解を深めるための具体例や比較対象を入れているか？ (有名人の例、使用シーンなど)

[ ] 内部情報の排除: 「前回の余り」など、作り手しかわからない内輪向けの情報は削除したか？

[ ] 情報の一貫性: 動画内で提示する情報に矛盾や不自然な点 (例:「居酒屋なのに車で行く」)はないか？

[ ] ネガティブ表現の回避: 「〜に注意」よりも「〜で良くなる」など、ポジティブな表現を心がけているか？

[ ] ターゲットを狭めすぎない: 「〇〇な人だけ見て」のような、過度にターゲットを限定する表現を避けているか？

[ ] CTA（行動喚起）の設置: 動画の最後に、視聴者にしてほしいアクションを明確に伝えているか？

[ ] 動画時間の最適化: 伝えたい内容に対して、動画の長さは適切か？ (長すぎていないか)

### 第3章：撮影編 (31-45)

[ ] 手ブレの防止: 三脚やジンバルを使うなど、手ブレを最小限に抑えられているか？

[ ] 明るさの確保: 画面全体が暗くならず、被写体がはっきりと見える十分な光量があるか？

[ ] ピントの確認: メインの被写体にしっかりとピントが合っているか？

[ ] 画質の担保: 高画質で撮影できているか？ (レンズの汚れ、設定の確認)

[ ] アングルの多様性: 同じ角度ばかりでなく、様々なアングル (寄り、引き、俯瞰など) で撮影しているか？

[ ] 背景の整理: 被写体の魅力が伝わるよう、背景は整理整頓されているか？ (生活感、不要な映り込み)

[ ] 背景と被写体のコントラスト: 背景と被写体が同化して見えにくくなっていないか？

[ ] 水平・垂直の意識: 画面の傾きがなく、安定した構図になっているか？

[ ] 視点移動の配慮: 視聴者が酔ってしまうような、急激なカメラワークは避けているか？

[ ] シズル感の演出: 料理や商品の魅力が最大限に伝わるような工夫 (湯気、音、光など) はあるか？

[ ] 人物の表情: 人物が映る場合、アカウントの世界観に合った魅力的な表情か？

[ ] 清潔感の担保: 映り込む手や服装、小物などに清潔感はあるか？

[ ] 撮影素材のバリエーション: 編集で使えるよう、十分なカット数とバリエーションを撮影したか？

[ ] 音の収録環境: アフレコや現場の音を綺麗に収録できる環境か？ (雑音の確認)

[ ] 画面の明るさの統一: シーンごとに明るさが極端に変わっていないか？

### 第4章：編集（映像）編 (46-60)

[ ] カットのテンポ: カット割りが細かく、テンポが良いか？ (間延びしているシーンはないか)

[ ] 不要部分の削除: 「えーっと」などの言い淀みや、無駄な「間」はカットされているか？

[ ] 視覚的な変化: ズームイン・アウトやエフェクトなどを活用し、視聴者を飽きさせない工夫があるか？

[ ] 映像と情報のリンク: 映像と、テロップや音声で伝えている内容が一致しているか？

[ ] 色味の調整: アカウントのトンマナに合った色味に調整・統一されているか？

[ ] トランジション（場面転換）: シーンの切り替えは自然で、動画のリズムを良くしているか？

[ ] 視線誘導: 視聴者の視線が分散しないよう、見せたいポイントに集中できる構成になっているか？

[ ] 強調の工夫: 最も伝えたいシーンは、スローモーションやエフェクトで強調できているか？

[ ] 情報の整理: 1つの画面に情報を詰め込みすぎていないか？

[ ] 人感・属人性の演出: 可能であれば、人が実際に使っている・食べている様子などを入れられるか？

[ ] 冒頭の掴み: 最も魅力的なシーン (完成品など) を冒頭に配置できているか？

[ ] ループ再生への配慮: 何度も見たくなるよう、動画の終わりと始まりが自然につながるか？

[ ] セーフティーゾーンの意識: アイコンやボタンと重なる部分に、重要な情報が配置されていないか？

[ ] 不自然な演出の排除: 違和感のある演出 (不自然な逆再生など) はないか？

[ ] 最終チェック: 全体を通して見た時に、リズムやストーリーに違和感はないか？

### 第5章：編集（テロップ）編 (61-80)

[ ] 視認性の高い配置: テロップがUI（いいねボタン、アカウント名など）と重なっていないか？

[ ] 読みやすいフォント: 動画の雰囲気に合い、かつ可読性の高いフォントを選んでいるか？

[ ] 適切な文字サイズ: 小さすぎて読めなくないか？大きすぎて圧迫感を与えていないか？

[ ] 背景とのコントラスト: 文字が背景に埋もれないよう、縁取りや背景色で工夫しているか？

[ ] 1画面の文字数: 視聴者が一瞬で読める文字数に収まっているか？

[ ] 表示時間の調整: 文字を読むのに十分な表示時間 (長すぎず短すぎず) が確保されているか？

[ ] テキストスタイルの統一: フォントの種類や色、大きさに一貫性があり、デザインが統一されているか？

[ ] 誤字脱字のチェック: 誤字脱字はないか？

[ ] 言葉遣いと表現: ターゲット層に響く、適切で魅力的な言葉を選べているか？

[ ] 専門用語の回避: 誰にでも理解できる平易な言葉で書かれているか？

[ ] 句読点・改行の適切さ: 文章が読みやすくなるよう、句読点や改行の位置は適切か？

[ ] テロップと音声の同期: テロップが表示されるタイミングは、音声（ナレーション）と合っているか？

[ ] 重要な部分の強調: キーワードや重要な数字は、色や大きさで強調できているか？

[ ] 情報量の最適化: テロップを入れすぎて、映像の邪魔になっていないか？

[ ] 視線の流れ: テロップの位置が頻繁に変わるなど、視聴者の視線が混乱する配置になっていないか？

[ ] サムネイルテキスト: 冒頭のテキストは、サムネイルとして表示された際にも魅力的か？

[ ] 漢字とひらがなのバランス: 漢字が多すぎて堅い印象になっていないか？

[ ] キャプションへの誘導: 動画内で伝えきれない情報は、キャプションで補足するよう促せているか？

[ ] 情報の補足: 映像だけでは伝わりにくい情報を、テロップで効果的に補足できているか？

[ ] 最終確認: 音を消した状態でも、テロップだけで大まかな内容が理解できるか？

### 第6章：編集（音声・BGM）編 (81-90)

[ ] BGMの選定: 動画のテーマやターゲット層の好みに合ったBGMを選んでいるか？

[ ] 音ハメ: 映像の切り替わりや動きが、BGMのリズムや盛り上がりに合っているか？

[ ] 音量バランス: BGMが大きすぎてナレーションが聞こえにくくなっていないか？

[ ] ナレーションの明瞭さ: ナレーションは滑舌が良く、聞き取りやすいか？

[ ] 声のトーン: ナレーションの声のトーンは、アカウントの世界観と合っているか？

[ ] 効果音（SE）の活用: 視聴者の注意を引くため、効果音を適切に活用できているか？

[ ] AI音声の質: AI音声を使用する場合、信頼性を損なうような機械的な声になっていないか？

[ ] 著作権の確認: 使用する音源は、著作権フリーか、適切なライセンスを取得しているか？

[ ] 無音部分の排除: 不要な無音部分はなく、音声が途切れていないか？

[ ] 音声と映像の同期: 発言内容や効果音が、映像のタイミングと正確に合っているか？

### 第7章：投稿・最終確認編 (91-100)

[ ] キャプションの最適化: 視聴者が読みたくなる、有益で分かりやすいキャプションか？

[ ] ハッシュタグの選定: 投稿内容と関連性が高く、効果的なハッシュタグを複数選べているか？

[ ] カバー画像（サムネイル）の設定: フィード画面で映える、魅力的なカバー画像を設定したか？

[ ] アカウントへの誘導: キャプション内でプロフィールへの誘導（@メンション）はできているか？

[ ] コメントへの返信準備: 投稿後のコメントや質問に、迅速かつ丁寧に対応する準備はできているか？

[ ] 誤情報の最終確認: 動画内のテキスト、音声、キャプションに、事実と異なる情報はないか？

[ ] CTAの再確認: 視聴者にしてほしいアクション（保存、コメントなど）をキャプションでも促せているか？

[ ] 外部ツールへの誘導（任意）: ストーリーズなどを活用し、リール投稿を告知する準備はできているか？

[ ] スマホでの表示確認: 実際にスマートフォンでどのように見えるか、最終プレビューで確認したか？

[ ] 投稿後の分析: 投稿後はインサイトを確認し、次回の改善に繋げる意識を持っているか？
