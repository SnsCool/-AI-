# 🏁 チームスローガン（Team Slogan）
**「共楽成進 〜ネガポジね。〜」**

**共に楽しみ、成長し、前に進む。**

どんな状況も“ネガをポジに変換する力”を全員で育てる。
### スローガンが意味すること
- **「共」：孤独にしない、一緒に進む**
- 成果が出ているときも、苦しいときも「チーム戦」として捉える
- できない人を責めるのではなく「一緒にどう乗り越えるか」を考える
- 個人プレーより「助け合って勝つ」を大事にする
- **「楽」：楽しむ姿勢を忘れない**
- 結果だけでなく「プロセスを楽しむ工夫」をする
- 雑談・ネタ・小さな共有を歓迎し、場の空気を明るくする
- 忙しいときほど、ユーモアと遊び心を大切にする
- **「成」：成長に貪欲**
- できないことや失敗を「伸びしろ」として歓迎する
- フィードバックを素直に受け取り、すぐ試してみる
- 昨日の自分を1ミリでも超える意識を持つ
- **「進」：挑戦から逃げない**
- 不安や怖さがあっても「一歩だけ」前に出てみる
- 新しいやり方・提案・チャレンジを応援する
- 結果より「挑戦したこと」をまず称賛する
- **「ネガポジね」：ネガティブをポジティブに言い換え、行動に活かすカルチャー**

# 🚫 チームの“やらないこと”（NOT TO DO）
### 1. 否定しない（No-Deny）
- 価値観・意見・行動を頭ごなしに否定しない
- 違いは「ズレ」として受け取り、攻撃しない
- 批判ではなく「提案」で返す
- 「なんでそう思ったの？」と一度相手の背景を聞く
**こんな言い換えを大事にする**
- ×「それは違う」→ ○「こういう見方もあるかも？」
- ×「意味わからない」→ ○「もう少し詳しく教えてほしい」
- ×「無理じゃない？」→ ○「どうしたら近づけそう？」
### 2. ネガティブにしない（No-Nega）
- 愚痴・悪口・裏での不満はゼロ
- マイナス感情は“事実と解釈”に分解して建設的に共有
- 失敗→「学び」として扱う文化を徹底
- その場で言えない不満は、落ち着いてから正面から伝える
**ネガをポジに変えるコツ**
- 事実：起きたことは何か？
- 解釈：自分はどう意味づけしているか？
- 変換：「この経験から得られるものは何か？」
### 3. 放置しない（No-Ignore）
- 相談放置、未読スルーはチームの信頼を削る
- 困っている人を置き去りにしない
- 返信は「速さより誠意」を意識して返す
- すぐに返せない場合は「一言リアクション」だけでも返す
**返信ガイドライン（目安）**
- 24時間以内に、何かしらリアクションを返す
- 深く返せないときは「後でちゃんと返すね」だけでも伝える
- 困っていそうなメッセージは、誰かが必ず拾う文化にする

# 🎯 チームミッション（Mission）
**「魂磨き」**
営業だけではなく、**人生のステージを一段上げる“自己進化の場”になること。**
### ミッションに込めた想い
- 兼任メンバーが多いからこそ、学びが交差する“化学反応の場”にする
- スキルだけでなく「人としての力（人間力・姿勢・覚悟）」を高め合う
- 困難を一緒に乗り越えることで“魂が磨かれる”チームにする
- お互いの人生のターニングポイントとなるような成長を実現する
### 行動レベルへの落とし込み
- 毎週1回、お互いの「成長ポイント」を言語化して共有する
- 小さな一歩でも称賛し、成長の積み重ねを可視化する
- ピンチのときこそ最も支え、最も伸びる瞬間と捉える
- 「しんどい」と言える空気をつくり、言ってくれた人を大事にする

# 📘 チーム内ルール（オペレーション基準）
### 1. 各自の部屋への投稿（なんでもOK）
- 活動報告、学び、気づき、雑談、喜び…なんでも投稿OK
- 投稿することで“互いの熱量”を可視化する
- スローガンの「共楽」を体現する場所として活用
- 「迷ったら、とりあえず書いてみる」を合言葉にする
**投稿のアイデア例**
- 今日の一言（日報・気づき・学び）
- うまくいったこと・うまくいかなかったこと
- 印象に残った一言やメモ
- 推しの本・動画・コンテンツ紹介
### 2. レオを悲しませない
（※レオ＝チームの芯・カルチャーの中心人物という意味合いで表現）
- チームの空気を悪くする行動をしない
- 自分の行動がチームに与える影響を意識する
- 関わる人を大切にし、仲間として尊重する
- 「これをしたらレオは喜ぶかな？」を行動基準にしてみる
### 3. 相談・報告は“早め・丁寧・本音”
- 早めに相談 → 誤解が減り、解決が速くなる
- 丁寧に共有 → 相手の理解と動きがスムーズ
- 本音で話す → すぐに改善につながる
**相談・報告のテンプレ**
- 【結論】今どういう状態か？
- 【背景】なぜそうなったか？
- 【相談】どうしたいか／決めきれないポイントはどこか？
### 4. フィードバック文化の徹底
- 褒める：事実ベースで、具体的に伝える
- 改善：人格否定ではなく、行動の指摘にフォーカスする
- 受け取る：すぐ反論せず一度“飲み込む”
- 与える：仲間の成長を願って必ず伝える
**フィードバックの例**
- 良かった点：「◯◯のときの△△がすごく良かった」
- 改善点：「□□の場面では、◇◇するともっと良くなりそう」

# 📅 3ヶ月目標（初評価期目標）
### 1ヶ月目：お互いを知る（関係構築期）
オンライン中心だからこそ、最初の1ヶ月は**“価値観理解”に振り切る。**
**具体アクション例**
- 自己紹介＋価値観シートの共有
- 実は大切にしている“人生のテーマ”を語る
- 得意・苦手・理想の働き方を共有する
- 趣味・プライベートの話もOK（距離が縮まる）
- 1on1や少人数トークで「深めの話」をする
### 2ヶ月目：伝え合い改善できる関係（信頼構築期）
言いたいことを率直に伝え、言われたことを素直に受け取れる**“相互信頼”**がテーマ。
**具体アクション例**
- 週1のフィードバックタイム（褒め×改善の2つ）
- 小さなモヤモヤは24時間以内に共有
- 情報の透明化（必要な情報はオープンに）
- 意見の違いは“対立”ではなく“成長のきっかけ”と捉える
- 「ありがとう」と「ごめん」をケチらない
### 3ヶ月目：成約率50%（成果創出期）
全員、成約率50%以上を達成。
ただ数字だけを追うのではなく、**再現性ある力を育てる。**
**具体アクション例**
- ロープレ・商談シェアを週2回実施
- 成功パターンを“型化”して全員に共有
- ボトルネックは全員で改善
- 商談1件1件を“魂磨き”の場と捉える
- 成約だけでなく「納得度」「安心感」も大切にする
