<summary>**研修期間の新メンバー（固定報酬型）の記載方法**</summary>
研修期間中のメンバーについては**、【20成約まで報酬が-20％】**となります。
そのため、請求書の作成は下記ルールに従ってお願いいたします。

1.　20成約までは** 固定給20万円＋歩合（研修レート）**
2.　達成後は **固定給25万円＋通常歩合**
3.　添付画像を参考に、着金順に「研修ナンバー」を付与してください。
　　例：（研修1）、（研修2）
4.　単価は**【固定報酬制＿研修期間】の金額**を記載してください。
5.　「営業固定報酬」の項目を作成し、単価を
　　**・**20成約までは** 200,000円
　　・**20成約達成した次月から**250,000円
　　**に設定してください。
6.　固定給を超えた場合は「営業固定報酬」の下（赤色部分）に**【固定報酬制＿研修期間】
　　**の金額を記載してください。
　　※着金順に計算
7.　研修期間中の請求書は、必ず**MG（チーム無所属の場合は教育担当者）にディスコードの個別DM** でもご提出ください。

<summary>**成約率30%以下の場合のインセンティブについて**</summary>
成約率30%以下のメンバーについては**、【基本のインセンティブ−1万円】**となります。
そのため、請求書の作成は下記ルールに従ってお願いいたします。

1.　成約率はローンチではなく、月単位で管理
  　 各マネージャー（チーム無所属の方は山口）から30％を切った場合は毎月10日までに
　   ご連絡いたします。
2.　単価は顧客ごとに **−1万円した金額** を記載してください。
3.　研修中の方の単価は**【フルコミッション制＿研修期間（30％未達）】**を記載してください。
 　 
4.　固定報酬型の方は歩合給の部分のみ、**【固定報酬制＿研修期間（30％未達）】**を記載してください。
5.　備考欄に「※成約率30％未達につき、次月の改善対象です」と記載してください。
