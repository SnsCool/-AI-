## 4. 赤字化する継続率（損益分岐の継続率）
**条件**：「顧客あたり最終利益 = 0」となる月間継続率を求める。
近似：**期待購入回数 ≈ 1 / (1 − 月間継続率)** を用いる。
- **A) COC=6,000**
3,280×(期待回数−1) = 5,220 → 期待回数 ≈ 2.59 → 1/(1−r)=2.59
⇒ **r ≈ 61.4%** を下回ると赤字（※既出）
- **B) COC=5,000**
3,280×(期待回数−1) = 4,220 → 期待回数 ≈ 2.29 → 1/(1−r)=2.29
⇒ **r ≈ 56.3%** を下回ると赤字
> 目安：月間継続率60%台前半が損益の分岐域。回収速度（2回更新で回収）を維持できるかが鍵。
