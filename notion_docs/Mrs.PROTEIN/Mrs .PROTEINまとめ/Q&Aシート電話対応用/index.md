# 飲み方・使い方
<details>
<summary>Q. どのように飲むのですか？</summary>
1包（20g）を200ml程度の水や牛乳、豆乳などに溶かしてお召し上がりください。コップに液体を入れ、粉末を加えてスプーンで粉を潰すようにかき混ぜると、ダマにならずにきれいに溶けます。
<details>
<summary>Q. シェイカーは必要ですか？</summary>
いいえ、Mrs.PROTEINは溶けやすさにこだわっているため、シェイカーは不要です。コップとスプーンがあれば、いつでもどこでも手軽にお作りいただけます。もちろん、シェイカーをお使いいただくと、より一層なめらかに混ざり美味しくお召し上がりいただけます。
<details>
<summary>Q. いつ飲むのがおすすめですか？</summary>
朝食の置き換えとしてお飲みいただくのが最もおすすめです。忙しい朝でも手軽にタンパク質やその他の栄養素をバランス良く摂取できます。その他、運動後30分以内のゴールデンタイムや、小腹が空いた時のおやつ代わり、就寝前の栄養補給など、ライフスタイルに合わせてお好きなタイミングでお召し上がりください。
<details>
<summary>Q. 1日何回飲めばいいですか？</summary>
1日1〜2杯を目安にお召し上がりください。Mrs.PROTEINは食品ですので、お食事とのバランスを考えて摂取量を調整してください。過剰な摂取は栄養バランスを崩す可能性があるため、お控えください。
<details>
<summary>Q. ホットで飲んでも大丈夫ですか？</summary>
はい、ホットでも美味しくお召し上がりいただけます。ただし、タンパク質は熱によって固まる性質があるため、熱湯を直接注ぐとダマになりやすくなります。人肌程度のぬるま湯で一度溶かしてから、温かい牛乳などを加えていただくのがおすすめです。

はい、問題ありません。牛乳や豆乳で割ると、よりクリーミーで満足感のある味わいになります。その他、アーモンドミルクやヨーグルト、スムージーに混ぜるのもおすすめです。
