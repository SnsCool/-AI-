# 【生徒向け画面】タスク管理アプリ　ジョーズくん　使い方

同期日時: 2026-07-01T16:44:14+09:00

---

ログイン
    1.ユーザー名を入力
    2. ログイン をクリック
    3. 昨日・今日・明日のタスクが表示されます
日報提出
    1. 日報入力画面
    「振り返りと今日の意気込み」を記入します。
    2. 日報入力
    「記入完了・コピー」を押します。
    （クリップボードにコピーされている状態になります）
    3. Discordに貼り付けてMGへご連絡ください
    Discordに貼り付けて、MGをメンションしてご連絡下さい。
    4.日報提出済み
マイページ
現在の投稿状況の確認やプロフィール設定ができるページです。
このページでは、セッションの履歴を確認でき、MGがセッションで話したことなどを確認することもできます。（セッション履歴）
    1. マイページ
    2. トグルをタップして開閉し、内容が確認できます
    3.目標は自身で入力しましょう
    4.セッション履歴
    5.マイルストーン達成バッジ
（上3つは自分でタップ）
タスクの追加方法
コーチングセッションでMGと決めたものに加えてタスクを追加したい場合は下記の手順追加ができます。
    1. カレンダー右上の「追加」をタップ
    2. 日時の選択をし、タスクを選択
    3. 「追加する」をタップ
    4. 追加完了
タスクの完了方法
    1. タスク右の「開始」をタップ
    2. 学んだことを入力
    3. 「完了」をタップ
    4. タスクが緑色になります
タスク期日の延期方法
    1. タスク右の「繰延」をタップ
    2. 繰延先の日時を指定する
    3. 「繰り延べる」をタップ
