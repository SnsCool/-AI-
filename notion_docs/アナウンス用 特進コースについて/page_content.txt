特進コースの目的と目標数値を記載する
特進コースとは
毎日SnsClubに向き合い、万垢を達成するためのコース
です。
万垢達成のために必要なこと
1投稿目をとにかく早く投稿する
  半年の中で、1投稿目が早いと　投稿期間が伸びる　→より多くの投稿を作り、
    動画編集のスキルが磨かれる
    フォロワーの求めているものに対して分析できる回数が増えることで
特進コース入会後の流れ
## 生徒のフロー
入会
└─→ スターター（step0 特進コースについての説明動画視聴・自己分析セッションの予約完了）
　-→アサイン完了後、MGのスレッドへ案内される
　-→（18STEP / 10日）
      └─→ 通常運用（以下のサイクルを繰り返す）

期間開始 
   ├─→ ◆自己分析セッション　（自己分析セッション担当者）
   │    ・自己分析シートを埋める
   │    
   ├─→ ◆2回目セッション（MG顔合わせ・目標設定セッション）
   │    ・数値KPI設定（収益/投稿数/フォロワー数）
   │    ・1週間のタスク決定（stepいくつまでSP完了させるか）
   │    ・週次タスクシート　埋める 
   │    ・1投稿目作成完了日（添削完了したもの、いつでも投稿できる状態のものを完成させる）
   │
   ├─→ ◆セッション（1ヶ月目：週1回、2,3,4ヶ月目：2週回、5,6ヶ月目：週1回/. 合計12回）
   │    ① 前回のセッションからの振り返り（1週間、2週間、1ヶ月の実績）
   │    ② 次2週間の「週次タスクシート」作成
   │       → 投稿目標本数やその他アクションを設定
   │
   ├─→ 日常行動（週次タスクシートに従う）
   │    ・今日の時間ブロックを確認 → タスクを完了
   │    ・投稿ワークフロー（10工程/1投稿）
   │    ・定期タスク（ラジオ/エスキャン/週報等）
   │
   └─→ 期間終了 → 振り返り（月初1〜10日でMGと一緒）
        ・投稿状況の「結果/要因/改善案」
        ・その他の「結果/要因/改善案」
        ・数値KPI結果を入力
特進コース入会の後の流れを記載する。
