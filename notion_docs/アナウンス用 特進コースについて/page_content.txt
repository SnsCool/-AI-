特進コースの目的と目標数値を記載する
特進コースとは
毎日SnsClubに向き合い、万垢を達成するため、計画的に行動するためのコース
です!!!!
万垢達成のために必要なこと
  ①1投稿目をとにかく30日以内に作り終えましょう
②1投稿目【作成完了】後、【10投稿完了】までを30日以内に完了しましょう
③半年間で100投稿を目指しましょう
なぜ？：万垢達成したアカウントは、30日以内に投稿を完了し、その後のアクションとしてもすぐに高頻度での投稿をしています！
  1投稿目が早いことのメリット
    投稿期間が伸びる　→より多くの投稿ができる
→そうすると：
    時間的余裕が生まれる　→アカウントや投稿自体の方向性の軌道修正や結果に対する改善できるからどんどん挑戦できる
→そうすると：
  半年で100投稿もの投稿を作るメリット
    圧倒的な投稿量は、圧倒的な経験に変わりあなたの味方になる　→成果を出す投稿が作れるようになり、継続のモチベーションにもなる
→そうすると：
    計画的に行動を進める環境で学習を進められる　→宣言したことをやり切った経験が自信になる
→そうすると：
1投稿目作成完了をSP開始から30日以内にやりきりましょう
  約束①：SnsClubでの半年間は、あらゆる時間をSnsClubの勉強に費やしましょう
  ・半年だけ、人生を変えるために、今までとの時間の使い方を変える努力をしましょう
・SnsClubに入って、たとえどんなに優秀な講師に質問や方向性を確認できる環境が手に入ったとしても、「生徒自身の投稿へのアクション」なしにはアカウントは育ちません。
・はじめは、投稿を作る上で物理的にどうしても時間はかかります。
・半年で「アカウントの運営において自走できる状態」を手に入れるためには初心者だからこそ、時間をかけましょう
    →コーチング担当者と、目標とタスクを明確にして約束する時間を「コーチングセッション」と呼びます。
    「やり切るタスク」と「納期」の約束をし、それを120%達成できるよう行動しましょう！！！
半年で11回、
  約束②：まずは自分で40点まで仕上げ、100点を目指すのはプロを頼りましょう
  ・何事も、はじめから100点を出すのは難しいです。ただし、「生徒の”0から1を生み出す”工程」なしには講師もMGもサポートができません。まずはやってみましょう!!!!
・やるべきことに対して行動するから課題や疑問が生まれます。これは、「行動できている証拠」です！！！まずは、「やるべきこと」に真っ直ぐ進んでいきましょう。
・
・半年で「アカウントの運営において自走できる状態」を手に入れるためには初心者だからこそ、時間をかけましょう
  →自己分析は、「自己分析担当者」が一緒に深めます
  ②1投稿目【作成完了】後、【10投稿完了】までを30日以内に完了しましょう
特進コース入会後の流れ
## 生徒のフロー
入会
└─→ スターター（step0 特進コースについての説明動画視聴・自己分析セッションの予約完了）
　-→アサイン完了後、MGのスレッドへ案内される
　-→（18STEP / 10日）
      └─→ 通常運用（以下のサイクルを繰り返す）

期間開始 
   ├─→ ◆自己分析セッション　（自己分析セッション担当者）
   │    ・自己分析シートを埋める
   │    
   ├─→ ◆2回目セッション（MG顔合わせ・目標設定セッション）
   │    ・数値KPI設定（収益/投稿数/フォロワー数）
   │    ・1週間のタスク決定（stepいくつまでSP完了させるか）
   │    ・週次タスクシート　埋める 
   │    ・1投稿目作成完了日（添削完了したもの、いつでも投稿できる状態のものを完成させる）
   │
   ├─→ ◆セッション（1ヶ月目：週1回、2,3,4ヶ月目：2週回、5,6ヶ月目：週1回/. 合計12回）
   │    ① 前回のセッションからの振り返り（1週間、2週間、1ヶ月の実績）
   │    ② 次2週間の「週次タスクシート」作成
   │       → 投稿目標本数やその他アクションを設定
   │
   ├─→ 日常行動（週次タスクシートに従う）
   │    ・今日の時間ブロックを確認 → タスクを完了
   │    ・投稿ワークフロー（10工程/1投稿）
   │    ・定期タスク（ラジオ/エスキャン/週報等）
   │
   └─→ 期間終了 → 振り返り（月初1〜10日でMGと一緒）
        ・投稿状況の「結果/要因/改善案」
        ・その他の「結果/要因/改善案」
        ・数値KPI結果を入力
