# KSF・転用AP

# KFF・改善AP
<details>
<summary>KFFとは</summary>
KFF（Key Failure Factors）とは、組織やビジネスが失敗する原因となる重要な要素や条件を指します。これらは、成功を妨げるリスクや障害となる要因であり、企業が目標を達成できない理由を明らかにするために分析されます。
<details>
<summary>改善APとは</summary>
KFFを受けて、同じ失敗を繰り返さないためにどうするべきか？というアクションプランを書き出してください
---
### 結果
- 講師３名の投稿がスパム判定で削除されてしまった
→削除後に別の投稿をしても再び削除され、アカウントステータスに影響が出た講師もいた

### 要因
- 教育およびローンチ時のクリエイティブ内に引っかかるワードが含まれていた
→稼ぐ、特典、無料、プレゼントなどなど

- 日常的に「特定コメント訴求」を行っていた
→普段から「意図的にエンゲージメントを上げている」と目をつけられていた可能性がある
## 改善AP
- 教育およびローンチ時のクリエイティブ内でスパム判定される可能性のあるワードを避ける
- 講師自身がリスクを負えるなら「特定コメントからフリーコメントに変更した上で」コメント誘導を行う
- リスクを負えない場合はプロフィール誘導を実施する
- チャンネル機能を活用してさらなる流入を狙う
- 普段からプレゼント企画等で特定コメント誘導を行っている講師には「危険性」を伝える
