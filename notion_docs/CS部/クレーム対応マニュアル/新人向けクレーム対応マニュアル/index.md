## はじめに
このマニュアルは、SnsClubのクレーム対応を初めて行う新人スタッフの方向けに作成されています。
### このマニュアルの使い方
1. **最初は順番に読む**：全体の流れを理解しましょう
1. **実際の対応時は該当箇所を参照**：慣れてきたら過去事例集を直接参照
1. **困ったらエスカレーション**：無理せず上司に相談しましょう
---
## クレーム対応の基本心得
### 🎯 5つの基本原則
### 1. 顧客第一の姿勢
- お客様の立場に立って考える
- 言い訳をせず、まず受け止める
- お客様の感情を理解しようとする
### 2. 迅速な初動対応
- **目標：気づいたら5分以内に初回返信**
- すぐに解決できなくても、まずは返信をする
- 「確認して後ほど返信します」でもOK
### 3. 誠実な謝罪
- 謝罪のレベルは感情に合わせる
- 言い訳を入れない
### 4. 明確なアクション
- 「何をいつまでにするか」を明確に伝える
- 曖昧な表現は避ける
- 約束したことは必ず守る
### 5. 記録と学習
- 全てのやり取りを記録する
- 同じ失敗を繰り返さない
- 良い対応例を共有する

## STEP1: クレームの分析方法
### 📊 3つの軸で分析する
### 1. 感情レベルの判定
### 🟢 レベル1：困惑・不安（軽度）
**キーワード：**
- 「不安です」
- 「分からない」
- 「確認したい」
- 「教えてください」
- 「困っています」
**お客様の状態：**
- 問題を理解できていない
- 解決方法を知りたい
- 少し心配している
**対応方針：**
- 丁寧に説明する
- 安心感を与える
- 軽度の謝罪
**例：**
```plain text
「フォロワー増加ランキングの集計方法が分からなくて不安です。
教えていただけますか？」
→ レベル1（困惑・不安）
```
---
### 🟡 レベル2：不満・失望（中度）
**キーワード：**
- 「不満です」
- 「残念です」
- 「期待外れ」
- 「物足りない」
- 「改善してほしい」
**お客様の状態：**
- サービスに失望している
- 期待値とのギャップを感じている
- 改善を求めている
**対応方針：**
- 誠実に謝罪する
- 具体的な改善策を提示
- 中度の謝罪
**例：**
```plain text
「講師のサポートが物足りないです。
もっと具体的なアドバイスが欲しいです。」
→ レベル2（不満・失望）
```
---
### 🔴 レベル3：怒り・憤慨（重度）
**キーワード：**
- 「怒っています」
- 「納得できない」
- 「損した」
- 「無駄だった」
- 「返金してください」
- 「退会します」
**お客様の状態：**
- 強い不満を感じている
- 金銭的損失を訴えている
- 継続を諦めている
**対応方針：**
- 深く謝罪する
- 即座にエスカレーション
- 返金も視野に入れる
- 重度の謝罪
**例：**
```plain text
「161,200円も払ったのに、この内容では納得できません。
返金してください。」
→ レベル3（怒り・憤慨）
```
---
### 2. カテゴリ分類
### 🔴 A. 返金要求系（最優先対応）
**A1. サービス内容不満による返金要求**
- サービスが期待外れ
- カリキュラムに不満
- 価値を感じられない
**判定ポイント：**
- 「返金」という言葉がある
- サービス全体への不満
- 金額の言及
**対応レベル：** 🔥 即座エスカレーション必須
**例：**
```plain text
「サービス内容が期待していたものと違います。
返金をお願いします。」
```
---
**A2. 講師対応不満による返金要求**
- 講師の指導品質に不満
- 1on1の期待値ギャップ
- 添削が不十分
**判定ポイント：**
- 「返金」+「講師」
- 指導内容への具体的不満
- 個別サポートへの言及
**対応レベル：** 🔥 即座エスカレーション必須
**例：**
```plain text
「講師の添削が適当で、個別指導が受けられませんでした。
返金をお願いします。」
```
---
**A3. サポート不備による返金要求**
- LINE案内が来ない
- サポートに連絡できない
- 質問に回答がない
**判定ポイント：**
- 「返金」+「サポート」
- 運営体制への不満
- コミュニケーション不備
**対応レベル：** 🔥 即座エスカレーション必須
**例：**
```plain text
「LINEの案内が来なくて、サポートが受けられません。
返金してください。」
```
---
### 🟡 B. 講師対応系（優先対応）
**B1. 講師変更要求**
- 講師との相性が合わない
- 運営方法への不満
- 別の講師を希望
**判定ポイント：**
- 「講師変更」という言葉
- 継続受講の意思あり
- 具体的な不満点
**対応レベル：** ⚠️ 24時間以内にエスカレーション
**例：**
```plain text
「現在の講師とは合わないので、
別の講師に変更していただけますか？」
```
**⚠️ 注意：契約書では第3回講義以降は講師変更不可**
---
**B2. 講師サポート不足**
- アカウント分析が不十分
- 具体的アドバイスがない
- 返信が遅い
**判定ポイント：**
- 「サポート」+「不足」
- 改善要望あり
- 継続意思あり
**対応レベル：** ⚠️ 24時間以内に対応
**例：**
```plain text
「講師からの具体的なアドバイスが少なくて困っています。
もっとサポートしてほしいです。」
```
---
**B3. 講師対応の質的問題**
- ホームルーム運営の問題
- 回答品質のばらつき
- 指導方法への疑問
**判定ポイント：**
- 指導の「質」への言及
- 期待値とのギャップ
- 他の受講生との比較
**対応レベル：** ⚠️ 24時間以内に対応
**例：**
```plain text
「ホームルームでの回答が曖昧で、
具体的なアドバイスが得られません。」
```
---
### 🟢 C. 運営・システム系（通常対応）
**C1. 運用ルール・規約関連**
- ランキング表示のルール
- 集計方法の確認
- 規約の理解
**判定ポイント：**
- 「ルール」「規約」
- 確認や質問
- 感情レベルは低い
**対応レベル：** ✅ 自動対応可能
**例：**
```plain text
「フォロワー増加ランキングは、
どのように集計されますか？」
```
---
**C2. 進捗管理・スケジュール**
- スタート時期の遅延
- 進捗共有が不足
- スケジュールへの不安
**判定ポイント：**
- 「進捗」「スケジュール」
- 不安の表明
- 状況確認
**対応レベル：** ⚠️ 24時間以内に対応
**例：**
```plain text
「グループ編成が遅れていて不安です。
いつ始められますか？」
```
---
**C3. グループ編成・初期設定**
- グループが未編成
- 初期設定が完了していない
- アカウント登録の問題
**判定ポイント：**
- 「グループ」「編成」
- 初期段階の問題
- 開始への影響
**対応レベル：** ⚠️ 24時間以内に対応
**例：**
```plain text
「グループがまだ編成されていません。
いつ確定しますか？」
```
---
### 3. 緊急度の判定
### 🔴 高緊急（即座対応）
**キーワード：**
- 「今すぐ」
- 「至急」
- 「早急に」
- 「法的措置」
- 「弁護士」
**対応：**
- **即座に上司へエスカレーション**
- 5分以内に初回返信
- 1時間以内に方針決定
---
### 🟡 中緊急（24時間以内）
**キーワード：**
- 「早めに」
- 「できるだけ早く」
- 「改善してほしい」
**対応：**
- 24時間以内に返信
- 担当者と相談して対応
---
### 🟢 低緊急（通常対応）
**キーワード：**
- 「いつか」
- 「そのうち」
- 「確認したい」
**対応：**
- 24時間以内に返信
- 通常の業務フローで対応
---
### 📝 分析の実践例
### 例1：
```plain text
「フォロワー増加ランキングに名前が載っていないのですが、
どうしてでしょうか？ 投稿もちゃんとしているのに不安です。」
```
**分析：**
- **感情レベル：** 🟢 レベル1（困惑・不安）
- 理由：「どうしてでしょうか？」「不安です」
- **カテゴリ：** C1. 運用ルール・規約関連
- 理由：ランキング表示のルール確認
- **緊急度：** 🟢 低緊急
- 理由：単純な確認、緊急性の言及なし
- **対応レベル：** ✅ 自動対応可能
---
### 例2：
```plain text
「講師の添削が適当で、161,200円も払ったのに納得できません。
返金してください。」
```
**分析：**
- **感情レベル：** 🔴 レベル3（怒り・憤慨）
- 理由：「納得できません」「返金」「金額の言及」
- **カテゴリ：** A2. 講師対応不満による返金要求
- 理由：「講師の添削」+「返金」
- **緊急度：** 🔴 高緊急
- 理由：返金要求、金額の言及
- **対応レベル：** 🔥 即座エスカレーション必須
---
### 例3：
```plain text
「現在の講師との相性が合わないので、
別の講師に変更していただけませんか？
継続受講は希望しています。」
```
**分析：**
- **感情レベル：** 🟡 レベル2（不満・失望）
- 理由：「相性が合わない」だが継続意思あり
- **カテゴリ：** B1. 講師変更要求
- 理由：明確な講師変更要求
- **緊急度：** 🟡 中緊急
- 理由：「できるだけ早く」は言及なし
- **対応レベル：** ⚠️ 24時間以内にエスカレーション

## STEP2: 応答文の作成方法
### 📝 応答文の基本構造
どのケースでも、以下の構造を守ります：
```plain text
【1. 挨拶】
お世話になっております。
SnsClub運営スタッフです。

お問い合わせいただきありがとうございます。

【2. 謝罪】
[感情レベルに応じた謝罪]

【3. 共感・理解】
ご連絡いただいた内容を拝読させていただき、
[不満/不安/疑問]をお感じになられたとのこと、
大変心苦しく思っております。

【4. 事実確認・詳細聴取】
つきましては、より適切な対応をさせていただきたく、
具体的にどのような点で[不満/不安/疑問]を
お感じになったのか、詳細をお聞かせいただけますでしょうか。

【5. 対応方針の提示】
いただいたご意見を真摯に受け止め、
[具体的な対応方針]について、
迅速かつ丁寧に対応させていただきたく存じます。

【6. 締めの挨拶】
お客様にご満足いただけるよう、
全力で取り組ませていただきます。

この度は、ご不便をおかけしてしまい、
心より深くお詫び申し上げます。

今後とも、より良いサービスをご提供できるよう
努めてまいりますので、何卒よろしくお願い申し上げます。
```

❌ **悪い例：**
```plain text
「改善に取り組んでまいります。」
```

```plain text
「講師への指導を本日中に行い、
```

- [ ] お客様の具体的な不満点に触れているか

- [ ] 敬語は適切か
- [ ] 上司の確認を得たか（必要な場合）

## STEP3: エスカレーション判定
### 🚨 エスカレーションとは？
上司や責任者に対応を引き継ぐことです。
**無理に自分で解決しようとせず、適切にエスカレーションすることが重要です。**
---
### 🔥 即座エスカレーション（5分以内）
以下の場合は、**即座に上司へ報告**してください：
### 必須条件
- ✅ 「返金」という言葉がある
- ✅ 感情レベルが「レベル3」（怒り・憤慨）
- ✅ 金額の具体的言及（161,200円など）
- ✅ 「弁護士」「法的措置」の言及
- ✅ 「退会」「解約」の明確な意思表明
- ✅ 複数回のクレーム（2回目以降）
### エスカレーション時の対応
1. **上司にすぐ報告**
- Slack/LINE/電話で即座に連絡
- 「返金要求が来ました」と明確に伝える
1. **お客様には一次対応のみ**
```plain text
お世話になっております。
```

## よくある失敗と対処法
### ❌ 失敗1: 言い訳をしてしまう
### 悪い例
```plain text
「申し訳ございません。
ただ、現在、講師の都合で...」
```
### 良い例
```plain text
「申し訳ございません。
運営として、このような事態を招いてしまい、
深く反省しております。」
```
### ポイント
- **言い訳は絶対にしない**
- 理由説明は後回し
- まず謝罪と受け止め
---
### ❌ 失敗2: 謝罪レベルが感情と合っていない
### 悪い例（怒りレベル3なのに軽度謝罪）
```plain text
お客様：「返金してください！」

返信：「ご不便をおかけしました。」
```
### 良い例
```plain text
お客様：「返金してください！」

返信：「この度は大変ご迷惑をおかけし、
心より深くお詫び申し上げます。」
```
### ポイント
- **感情レベルに合わせて謝罪の強度を調整**
- 怒りレベル3 → 重度謝罪
- 不満レベル2 → 中度謝罪
- 不安レベル1 → 軽度謝罪
---
### ❌ 失敗3: エスカレーションが遅い
### 悪い例
```plain text
「返金してください」というメッセージに対して、
自分で返金の判断をしてしまう
```
### 良い例
```plain text
「返金してください」というメッセージを見たら、
即座に上司に報告
```
### ポイント
- **返金要求は必ずエスカレーション**
- 自分で判断しない
- まず上司に相談
---
### ❌ 失敗4: 曖昧な返信
### 悪い例
```plain text
「改善に取り組んでまいります。」
```
### 良い例
```plain text
「講師への指導を本日中に行い、
明日までに改善策をご提示いたします。」
```
### ポイント
- **「いつまでに」「何をするか」を明確に**
- 曖昧な表現は避ける
- 具体的なアクションを示す
---
### ❌ 失敗5: お客様の言葉を無視
### 悪い例
```plain text
お客様：「具体的なアドバイスが得られない」

返信：「サポート不足で申し訳ございません。」
```

### 良い例

### ポイント
- **お客様の言葉をそのまま引用**
- 「ちゃんと読んでいる」という印象を与える
- 共感を示す

### 良い例

### ポイント
- **初動対応は5分以内を目標**
