# 領域別チェックリスト（配属と成果物つき）
**会場・機材（担当：会場リーダー）**
- 原状回復・点検表／破損・清掃・残置の報告
- 搬入出動線の渋滞分析→次回の時間割/人員配置案
- 機材台帳更新（必要台数・型番・ケーブル種・冗長案）
**飲食（担当：ケータリング）**
- 実食数、増減理由、ピーク時の列滞留時間→再配置・導線案
- メニュー評価（味/温度/補充頻度/アレルギー表記）→改善策
**プログラム・進行（担当：舞台監督）**
- 進行遅延ポイントと原因（転換/配線/登壇者移動）
- 事前リハのチェック項目アップデート（音声ライン/映像/スイッチャ）
**託児（担当：託児窓口）**
- 年齢帯別の需要/呼び出し件数/満足度→枠設定の最適化
- 体調・アレルギー対応の標準手順修正、持ち物リストの改善
- 個人情報の保存/廃棄スケジュールと責任者の明確化
**広報・UGC（担当：コミュニティ）**
- ハッシュタグ回収／優秀UGCのリポスト計画
- 参加者事後ナーチャー（次回予告・限定先行の案内）
**財務（担当：経理）**
- 請求・支払・入金の全件消し込み
- 収支分析：大項目（会場/機材/飲食/宿泊/人件費/託児）×差分要因
**法務/個人情報（担当：総務）**
- 画像・動画の二次利用可否台帳／モザイク対象の管理
- 託児・申込データの保存期間と削除証跡の管理
**ナレッジ（担当：PMO）**
- テンプレ更新（受付フロー、MC台本、場内サイン、緊急時対応、託児案内）
- リスクレジスター更新（発生確率×影響度×対策）
