# リール制作の作業フローについて

◼︎質問
リールを作る際に、一つ作って依頼→一つ作って依頼というように一つずつ作成するのか何本か一気に台本を作って、台本を書く日、動画を撮影する日というように作業で区切って作成するのかそれぞれのメリットやデメリットがあれば聞きたいです！まだ投稿までの段階ですが、いろいろ模索中なので、意見を聞いてみたいです！

◼︎回答
それぞれにメリット・デメリットがあります。一つずつ作成する場合のメリットは、講師からのフィードバックをすぐに次の投稿に活かせることです。特に初心者の方は、この方法で素早く改善サイクルを回すことをおすすめします。デメリットは作業効率が悪くなることです。一方、まとめて作業する場合のメリットは、作業効率が上がることです。同じ作業をまとめてやることで、集中力も保ちやすくなります。デメリットは、フィードバックを受けるまでに時間がかかることです。私のおすすめは、最初の10～20投稿は一つずつ作成し、慣れてきたら徐々にまとめ作業に移行していくことです。

# スケジュール管理について

◼︎質問

◼︎回答

# TikTokの身バレ防止対策について

◼︎質問
駒居さん、運営のみなさん、いつもありがとうございます。私は汚部屋脱却をテーマに投稿しています。伺いたいのはTikTokの身バレ防止についてです。既存のTikTokアカウントにアカウントを追加したら、知り合いが表示されています。このまま投稿を始めると、知り合いに投稿が表示されるのかと思い投稿できずにいます。投稿テーマの都合上、フォロワー数が増える前の身バレはできる限り避けたいです。後ろ向きな意見で申し訳ないのですが、初期の学習動画などでアドバイスいただけるとうれしいです。

◼︎回答

# 威圧的な人への対処法について

◼︎回答
質問者さんの「頼りにしてます」アプローチは素晴らしいと思います。私も基本的には同じ考えです。威圧的な人は、自分が認められたい、価値を感じたいという欲求が強い傾向にあります。そのため、まずは相手の価値を認めることから始めます。具体的には、「〇〇さんの意見、いつも勉強になります」「その視点は気づきませんでした」など、相手の発言を肯定的に受け止めます。その上で、「私の考えも聞いていただけますか？」と許可を求める形で自分の意見を伝えます。また、1対1ではなく、複数人での議論に持ち込むのも効果的です。威圧的な態度は、集団の中では和らぐ傾向があります。何より大切なのは、相手の攻撃性に巻き込まれないこと。冷静さを保ちながら、建設的な議論に導いていきましょう。

# 美容系投稿の撮影について

◼︎質問
美容系の発信をしようと考えており、もうすぐ初投稿です。動画を撮影するにあたり角度や写りなど気に入らないと何度も撮影して時間がかかってしまいます。撮影の時に意識していることやどれくらい撮影に時間をかけているのか教えて頂きたいです。

◼︎回答
美容系は確かに見た目の印象が重要なので、お気持ちわかります。ただ、完璧を求めすぎると投稿が進まなくなってしまいます。私が意識しているのは「70点でOK」ルールです。まず撮影時間は1投稿あたり30分以内と決めてください。撮影のコツは、事前に構図を決めておくことです。ロールモデルの動画を参考に、カメラの位置や角度を固定してから撮影を始めます。また、自然光が入る時間帯（午前中がおすすめ）に撮影すると、肌がきれいに映ります。最初から完璧を目指すのではなく、投稿を重ねながら徐々に質を上げていく意識が大切です。

# コメント誘導施策の是非について

◼︎質問
駒居さん、いつもありがとうございます！特典配布などを目的に「キーワードをコメント欄に入力してもらう」ことでエンゲージメント率を高める施策って、ここ最近SNS運用の中でもよく見かけるようになってきたと思います。ただ、少し前にInstagramのアダム・モッセーリ氏が「コメント誘導によるエンゲージメント獲得は、不自然な操作とみなされ、アルゴリズム的にペナルティの対象になる可能性がある」と発信していた件がありましたよね。そんな中で、SnsClubの中でも実際にこの手法を使っているアカウントも見受けられます。駒居さんご自身は、この施策についてどのようにお考えでしょうか？やはりリスクがある手法として控えた方が良いのか、それとも使い方次第で有効になり得るのか……。ご意見をお聞かせいただけたら嬉しいです。

◼︎回答
確かにアダム・モッセーリ氏の発言は重要な指針です。私の見解としては、「自然な流れでのコメント促し」はOK、「露骨な誘導」はNGと考えています。例えば「〇〇と入力してください」のような機械的な誘導は避けるべきです。一方で「あなたの経験を教えてください」「どちらが好きか教えてください」など、自然な会話の延長としてのコメント促しは問題ないでしょう。重要なのは、コメントを通じて本当の価値提供ができているかどうかです。特典目的だけの誘導は避け、コミュニティ形成や価値ある情報交換を目的としたコメント促しを心がけることをおすすめします。

# コミュニケーション苦手克服について

◼︎質問

◼︎回答

# フォロワー停滞期の対策について

◼︎質問
リール動画の方向性についてお聞きしたいです。私は今節約レシピを投稿しており、78投稿で2.4万人フォロワーがいますが、フォロワーの伸び率は停滞してきており、再生数も伸び悩んでいる状態です。そのような場合は一度バズっている内容をそのまま継続させた方が良いのか、少しコンセプトを変えて反応を見た方が良いのか、駒居さんならどうされるのか教えていただきたいです。よろしくお願いします。

◼︎回答
まず78投稿で2.4万人は素晴らしい成果です。停滞期に入ったと感じる場合、私なら「微調整」から始めます。完全にコンセプトを変えるのではなく、既存の「節約レシピ」の中で新しい切り口を試してみてください。例えば、100円以下レシピ、10分以内レシピ、材料3つだけレシピなど。また、トレンドの音源を使ったり、サムネイルのデザインを変えたりする小さな変化も効果的です。さらに、過去にバズった投稿をリメイクするのもおすすめです。大きな方向転換は、既存フォロワーが離れるリスクもあるので、慎重に判断しましょう。

# ポジティブ思考の身につけ方について

◼︎質問
いつもラジオを楽しく拝聴しています。駒居さんは常にポジティブで、素晴らしい思考の持ち主だと思います。私はわりとネガティブ思考なのですが、どのようにポジティブ思考を身につけられたのでしょうか？？元からなのでしょうか？？

◼︎回答
実は私も元々はネガティブ思考でした。変わったきっかけは「事実と解釈を分ける」ことを意識し始めたことです。例えば「投稿が伸びない」は事実ですが、「自分には才能がない」は解釈です。事実は変えられませんが、解釈は自分で選べます。また「できない理由」ではなく「できる理由」を探す癖をつけました。時間がない→隙間時間を見つける、お金がない→無料でできることから始める、という具合に。最初は意識的に行う必要がありますが、続けていると自然とポジティブな思考回路になっていきます。完璧にポジティブになる必要はありません。少しずつ思考の癖を変えていきましょう。

# アフレコと台本の調整について

◼︎質問
台本と構成とアフレコの関係について。台本と構成はいい感じでも、アフレコが上手く入らなくて、あちこち調整しまくらないと収まらなくなる。どれを基準にしたら良いか？

◼︎回答
これは多くの方が悩むポイントです。基準にすべきは「動画の長さ」です。まず目標とする動画の長さ（例えば30秒）を決め、それに収まるよう調整します。優先順位は、1.重要な情報を削らない、2.テンポを保つ、3.自然な話し方を維持する、です。具体的には、台本を音読して時間を測り、長すぎる場合は冗長な表現を削ります。「〜することができます」→「〜できます」のように短縮できる部分を探しましょう。それでも収まらない場合は、構成自体を見直す必要があるかもしれません。講師に相談しながら調整してください。

# プラットフォーム別の伸びの違いについて

◼︎質問
いつもお世話になっております。インスタより、TikTokやYouTubeの方が伸びがいいのは何かあるんでしょうか？分析が足りていないと思いますが聴いてみたかった事です。

◼︎回答
一般的にTikTokは最も伸びやすく、次にYouTube Shorts、Instagramという順になることが多いです。理由は、TikTokのアルゴリズムが新規アカウントにも比較的優しいことと、コンテンツの拡散力が高いことです。ただし、各プラットフォームで求められるコンテンツの特性が違います。TikTokはエンタメ性重視、Instagramは見た目の美しさ重視、YouTubeは情報価値重視の傾向があります。重要なのは、どこで伸びるかより、どこでマネタイズするかです。目的に合わせてメインプラットフォームを決めましょう。

# 自信作が伸びなかった時の対処法について

◼︎質問
駒居さん、いつもラジオありがとうございます！先日の「聞く力・質問力」についてのお話、とても勉強になりました。相手の言葉をただ受け取るだけじゃなく、「引き出す姿勢」が大切なんだと知って、SNSでの発信にも通じるものを感じました。そこで質問です。「これは絶対いける！」と思って投稿したものが全然伸びなかったとき、自信を失ってしまいます…。そういう時、駒居さんならどうやって気持ちを切り替えますか？見直しても原因がわからないとき、次に活かすヒントをどう見つけたらいいか悩んでいます。

◼︎回答
私も同じ経験が何度もあります。そんな時は「データと感情を分ける」ことを意識します。まず、伸びなかったのは事実として受け止めます。ただし「自分には才能がない」などの感情的な解釈は避けます。次に、必ず「3つの学び」を見つけます。例えば、サムネイルの工夫点、音源の選び方、投稿時間など。原因がわからない時は、競合と比較してみてください。自分の投稿と何が違うのか、細かく分析します。そして最も大切なのは「失敗は成功のデータ」と捉えること。伸びなかった投稿も、次の成功につながる貴重な経験です。落ち込む時間を分析の時間に変えましょう。

# プライベートの問題がSNS活動に与える影響について

◼︎質問
お世話になっております。最新のラジオでモチベーションについての内容拝聴しました。投稿やインスタとは直接関係しないことで自分が落ち込んだりモチベーションが下がっている時(不幸なことや悲しい事があった等)、周りにその影響を出さないよう気をつけると普段以上にコメントや質問が減ってしまい、そのことに焦りを感じる時があります。余計なことを考えず投稿に注力したいが、マイナスな出来事に気持ちが引っ張られてしまい、プラス思考になれないことが苦しく自分を責めてしまいます。インスタに注力できないのは成功に対する気持ちが弱いからだと思われますか。

◼︎回答
まず、落ち込むことがあっても当然です。人間ですから。成功への気持ちが弱いわけではありません。むしろ、そんな状況でも活動を続けようとしている時点で、強い意志を持っている証拠です。私がおすすめするのは「最低限ライン」を決めることです。例えば「どんなに辛くても1日1投稿だけはする」など。完璧を目指さず、継続を重視してください。また、辛い時こそ仲間に頼ってください。クラスメイトに「今ちょっと辛い時期で」と伝えるだけでも楽になります。無理に明るく振る舞う必要はありません。自分を責めず、できることから一歩ずつ進みましょう。
