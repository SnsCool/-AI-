# あぶらだあい

  ① 半年間の変化（スクラブに入ってからの変化・この半年の価値）
  「私のアカウントは私の実績みたいな」と捉え方が変わった。
→ 「本当？私の実績はこれですってな位置づけで考えています」と、アカウントを“積み上げた成果”として扱う意識が明確になった。
  収益の置き方も整理された。
→ 「アフィリエイトではなく運用の方に私は重きを持って行きたい」と、やりたい方向（仕事としての形）を優先したい気持ちが言語化できた。
  「頑張った」を自分で認められるようになった。
→／「仲のいい人たちには、いかに自分が頑張ったかをすごく言ってます」と、自分の努力を“成果として言える”状態になっている。
  この経験を「子育てにも還元したい」という価値観につながっている。
→ 「これをいい意味で子育てに還元してって、ママもこれだけ頑張ったから…」という形で、挑戦の意味づけが深まっている。
  ② 卒業後の目標（在宅勤務／1年後のなりたい姿）
  1年後のビジョンははっきりしている：
→ 「一応ビジョンとしては一年後に在宅勤務」
  具体的な働き方イメージ：
→ 「午前中だけ事務所出勤にして、あとの午後はもう在宅勤務」
→ 「子供達が帰ってくる時間には自宅にいたいっていうのがずっと目標」
  そのために、アカウントは“実績”として継続する前提：
→ 「自分のアカウントは実績として運用して継続していく」という整理。
