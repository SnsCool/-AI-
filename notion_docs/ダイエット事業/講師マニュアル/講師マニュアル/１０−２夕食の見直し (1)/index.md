# 夕食の見直し
## なぜ夕食を見直す必要があるの？
- 夕食は「寝る前に食べる最後の食事」。
- 内容やタイミングを誤ると **脂肪が蓄積・睡眠の質低下・むくみ** の原因に。
👉 ダイエット成功には夕食の見直しが欠かせません。

## 夕食の理想的なPFCバランス（目安）
- **たんぱく質：20g**（肉・魚100g／卵＋納豆など）男性は30g
- **炭水化物：ご飯100g前後（糖質40g程度）男性の場合は130gくらい**
- **脂質：10〜15g程度**
👉 昼より軽め、でも「抜かない」のがポイント。
---
## ダイエット中の夕食NG例
- 大盛りラーメン＋餃子＋チャーハン → 糖質＋脂質過多
- 焼肉で白ご飯大盛り＋ビール → 脂肪蓄積リスク大
- コンビニ弁当＋菓子パン → 塩分・脂質過多、栄養偏り
---
## 遅い時間に夕食を食べる場合
- **たんぱく質は必ず摂る** → 筋肉分解を防ぐため
- 昼〜夜の間隔が長いときは、間食で以下を補給すると良い
- サラダチキンor焼き魚or豆腐系の料理
- 小サイズおにぎり
- サラダ
👉 血糖値の急上昇を防ぎ、夕食の食べすぎを予防
---
## 就寝前にお腹が空いたときの工夫
- 空腹で眠れない場合は「軽め＆消化に良いもの」を選ぶ
- ホットミルク
- 無糖ヨーグルト
- プロテイン少量
- ゆで卵
- めかぶ
👉 血糖値を大きく上げず、睡眠の質を守れる
---
## 夕食と翌朝のつながり
- 夜に脂質・塩分が多い → 翌朝のむくみ・だるさ・体重増に直結
- 逆に、夕食を軽く整えると翌朝の目覚めや体調が快適に
👉 「夜の選択＝翌日の快適さ」と意識できると習慣化しやすい
