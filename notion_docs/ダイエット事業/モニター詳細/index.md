### サポート詳細
### **1. ポータルサイト**（Notion）
- ダイエットの基礎知識（食事・運動）
- よくある質問（食事メニュー、間食、運動法など）を集約。
- 「質問する前にまずここを見る」運用をオリエンで習慣化。
- フィットネス配信のアーカイブ視聴権利
### 2. 毎日の記録入力
- **フォーム入力ベース**
- Googleフォームを用意。
- 入力されたデータは自動でスプレッドシートに反映。
- スプレッドシートでは個人別タブを自動生成し、体重推移グラフも自動作成。
### 3. リマインド体制
- **メインツールはDiscordに統一**
- 朝：全体リマインド部屋 → 「今日の体重を計測してね！」
- 夜：全体リマインド部屋 → 「今日の1日の記録を入力してね！」
- **実装方法**
- 定型文をBotで自動送信（Zapier / Discord Botで対応可）。
### 4. サポート体制
- **週次レビュー**
- 各チーム講師がグループMTGで振り返り＆シェア。
### 5. 使用ツールの統一
- **Discord優先 / LINE最小化**
- 原則：Discordで全てのサポートを完結。
- 例外：決済・緊急事務連絡のみLINEを利用。
- **未入力者への対応**
- 3日連続未入力 → Discordでメンション。（一旦チーム講師の人力でOK）
- 反応がなければ、最終手段としてLINEで声かけ。
### 6. VIPプラン
- 月2回の個別ZOOMサポート（1回60分）
- Discordでの個別チャット無制限
- 相談しやすい専用スレッドを用意。

## **1｜再現性の検証**
### **✔ どのくらいの期間で成果が出るか**
- 1ヶ月・2ヶ月・3ヶ月の体重/体脂肪率の平均変化
- 行動量（食事記録数・運動数）と成果の相関
- どのタイプの人が最短で成果が出るか（生活習慣・体質）
### **✔ どのタイプの顧客なら成果が出やすいか**
- 食事改善ができるタイプ／忙しくて難しいタイプ
- 自己管理が得意か／不得意か
- 仕事・生活スタイル別の伸びやすさ
### **✔ 再現可能な「勝ちパターン」の抽出**
- 成果が出た人の1日の行動ルーティン
- 成果が出た人の食事の型（朝昼夜の食べ方）
- 成果が出るフィードバックの特徴（頻度・内容）
- 成果が出るAIリマインドの型
