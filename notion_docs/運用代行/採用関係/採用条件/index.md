## コンテンツ

# ◾️ディレクターの採用条件
1.採用する場合
・経験年数：SNS運用またはディレクション経験 1年以上
・実績アカウント：実績提出できる法人アカウントが3件以上or個人のフォロワー1.5万人以上
・働き方：週20時間以上の稼働が可能
・その他：平日9:00~18:00の間で3時間以内即レスできる
・報酬目安 ：1クライアントあたり最大月5万円（税込）
2.育成する場合
・企画出し台本からスタート
・3ヶ月はマネージャーと併走し、マネージャーがクライアントワークや指示を担当＆教育
・3ヶ月後にディレクターとしてデビューさせる

# ◾️企画・台本の採用条件
・経験年数：SNS運用の経験が6ヶ月以降
・実績アカウント：実績提出できる法人アカウントが3件以上or個人のフォロワー0.5万人以上
・働き方：週5時間以上の稼働が可能
・その他：平日9:00~18:00の間で3時間以内即レスできる
・報酬目安 ：1本あたり2,000円〜（税込）
# ◾️編集者の採用条件
・経験年数：SNS運用の経験が6ヶ月以降
・実績アカウント：実績提出できる法人アカウントが3件以上
・働き方：週5時間以上の稼働が可能
・その他：平日9:00~18:00の間で3時間以内即レスできる、プレミアプロを使用できる方優遇
・報酬目安 ：1本あたり2,000円〜（税込）
# ◾️撮影者の採用条件
・経験年数：SNS運用の経験が6ヶ月以降
・実績アカウント：実績提出できる法人アカウントが3件以上
・働き方：月5時間以上の稼働が可能
・その他：
・報酬目安 ：1回あたり15,000円〜（税込）
