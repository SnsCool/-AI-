# ベルラスダイエット食事法
①炭水化物（糖質）を控え、適量にする
②その分、良質なタンパク質・脂質をしっかり摂取する
③炭水化物・脂質・タンパク質が旨く代謝できるように、
ビタミン・ミネラル・酵素・水をしっかり摂る
④腸内環境改善のために、食物繊維をしっかり摂る
⑤細胞のアンチエイジングのために、ファイトケミカルを摂る
└イソフラボン・リコピン・ポリフェノール

細胞を傷つけ老化させる原因になる5つの不要物を控える
①過剰な糖質
②添加物
③アルコール
④トランス脂肪酸
⑤悪い油（酸化したもの）
## →必要栄養素9つをしっかり摂る＋5つの不要物を控えていく

太ってしまう食事の原因は2つ
- 糖質の摂りすぎ
- 糖質×脂質の摂りすぎ

痩せられない人の原因
- 筋肉減少・細胞劣化が進み基礎代謝量が低下している
- 活動量が少ない割に糖質・糖質×脂質を摂りすぎている
- 栄養バランスが悪く代謝が滞っている
- 添加物・悪い油・人工甘味料など体にとって不要なものを摂りすぎている

急激な体重減少が起こると、、、
体を守るためにエネルギー源として脂肪を溜め込む
エネルギーを確保するためエネルギーを消費する筋肉を削いでしまう
→ 恒常性機能（ホメオスタシス）により急激な変化から身を守るため一時的に痩せても元の状態に戻りやすい（リバウンド：体脂肪のみが増える）
→ リバウンドを繰り返すほどさらに痩せにくい体質になる
### →しっかり食べて筋肉や骨を残しつつ、なるべく体脂肪のみを燃やしていくことが重要

代謝を上げ太りにくい体質を作ること
→ そのためにはカロリーを減らし食事量を落とすのではなく、同じカロリーでもその内訳が大切な要素
