## 1. ローンチ戦略：段階的アプローチ
残り約50日という限られた時間で確実にローンチするため、以下の2段階で進めます。

**Phase 1: ソフトローンチ（12月中旬）**
- 目的: 既存会員30名にβ版を提供し、フィードバックを得ながらサービスを改善する。
- 対象: 既存マネタイズ講座の会員30名
- 内容:
- 講師: 1名（即戦力）
- 教材: パイロット動画2本 + 基礎教材
- サポート: 週1回のグループコンサルティングのみ
**Phase 2: 本格ローンチ（2026年1月下旬）**
- 目的: β版のフィードバックを反映し、SnsClub全体および外部に向けて本格的にサービスを展開する。
- 内容:
- 講師: 3〜5名体制
- 教材: 全15本の動画教材を完備
- サポート: エリートコースの1on1サポートを開始

## 4. 必要なアクション（TODOリスト）
**【今すぐやること】**
1.講師採用要件を確定する
- 必須条件: TikTok Shopで月100万円以上の収益を6ヶ月以上継続
- 歓迎条件: 教育経験、ライブコマースでの高額販売実績
2.講師の募集を開始する
- SnsClub内でのスカウト: 既存ネットワークから有望な人材に声をかける。
- SNSでの公募: XやInstagramで募集要項を公開する。
- リファラル採用: 既存講師からの紹介を依頼する。
3.β版のサービス詳細を決定する
- 料金: 既存マネタイズ講座と同じ（ライト¥10,000/月、エリート¥30,000/月）
- サポート範囲: 週1回のグルコン、チャットサポート（講師1名で対応可能な範囲）
- カリキュラム: パイロット動画2本 + 基礎的なテキスト教材

**【各週のアクション詳細】**
**Week 1: 講師採用と企画**
- 担当: 陸さん・凪
- アクション:
- 講師候補者リストを作成する。
- 候補者に個別に連絡し、面接のアポイントを取る。
- β版のサービス概要をドキュメントにまとめる。
**Week 2: 講師選考と教材準備**
- 担当: 陸さん・凪
- アクション:
- 1日3〜5名のペースでオンライン面接を実施する。
- 面接と並行して、動画2本（ロードマップ、商品リサーチ）の構成案を作成する。
**Week 3: 講師決定と教材制作**
- 担当: 陸さん、採用講師・凪
- アクション:
- 講師を1名決定し、クラウドサインで契約を締結する。
- 採用した講師と動画の台本を完成させる。
**Week 4: 教材完成とローンチ準備**
- 担当: 採用講師、動画編集者・凪
- アクション:
- 講師に動画撮影を依頼する（スマホ撮影でOK）。
- 撮影データを動画編集者に渡し、編集を依頼する。
- 陸さんは、既存会員30名への案内文（LINE、メール）を作成する。
**Week 5: ソフトローンチ**
- 担当: 陸さん・凪
- アクション:
- 既存会員30名にβ版を案内し、申込受付を開始する。
- 申込者と第1回グルコンの日程を調整する（12/20前後が目安）。
**Week 6-7: サービス開始と改善**
- 担当: 採用講師、陸さん・凪
- アクション:
- 第1回グルコンを開催し、会員のモチベーションを高める。
- 会員からの質問や要望を収集し、サービス改善リストを作成する。
- 本格ローンチに向けたタスクを洗い出す。
