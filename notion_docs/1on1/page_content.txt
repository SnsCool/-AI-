    # 台本作成の要件                                                                  
                                                                                      
    ## 登場人物                                                                       
    - **講師（speaker_id: "sp1"）**:                                                  
    専門知識を持つ教える側。分かりやすく丁寧に説明する。                              
    - **生徒（speaker_id: "sp2"）**:                                                  
    学習者として質問や確認をする側。視聴者の代弁者として疑問を投げかける。            
                                                                                      
    ## 📋 章構成（目次に従う - 最優先）                                               
                                                                                      
    以下の目次に従って、台本のセクションを構成してください：                          
    1. プロフィールの概要と重要性                                                     
    2. アイコンの選び方とポイント                                                     
    3. アカウント名・ユーザーネームの設定方法                                         
    4. ハイライトの活用方法                                                           
    5. プロフィール文章の書き方とベストプラクティス                                   
                                                                                      
    **重要**:                                                                         
    - 上記の章構成に従って台本を作成してください                                      
    - 各章の内容を充実させて、全体で10-20分程度を目標にしてください                   
    - 内容が豊富な章はしっかり時間をかけて詳しく説明してください                      
                                                                                      
    ## 台本の構成                                                                     
    1. **章構成**: 目次に従ってセクションを構成する（必須）                           
    2. **全体の長さ**: 10-20分程度を目標（内容を充実させる）                          
    3. **対話の比率**: 生徒の発話3に対し、講師の発話7の割合を維持                     
       - **講師は1回の発話でしっかりと説明する**（250-500文字、50秒-1分40秒）         
       - **生徒の質問は要点を絞る**（40-80文字、8-16秒）                              
       - **やり取りの頻度を減らし、講師の連続説明を増やす**                           
       - 生徒は重要なポイントで質問し、講師が詳しく答える流れ                         
    4. **自然な会話形式**で進行（堅苦しくない、親しみやすいトーン）                   
    5. **発話の長さガイドライン**:                                                    
       - 講師の詳しい説明: 250-500文字（連続した説明を心がける）                      
       - 生徒の質問: 40-80文字（簡潔に）                                              
       - 各セクションは2-4分程度を目安に                                              
    6. **具体例や詳細な説明**を豊富に含める                                           
    7. **内容の質と充実度**を重視                                                     
    8. **🆕【重要】長い発話（100文字以上）は parts に分割して変化をつける**           
       - 発話を意味のまとまり（段落、トピック）ごとに分割                             
       - 各partにtext と character_direction を含める                                 
       - 自然な流れ: 導入(medium) → 説明(medium) → 強調(large) のようなパターン       
       -                                                                              
    話者が交代する場合は必ず新しいdialogueにする（parts内では話者変更しない）         
                                                                                      
    ## 🎭 キャラクター演出の指定（重要）                                              
                                                                                      
    各dialogueには必ず **character_direction** を含めてください：                     
                                                                                      
    ### サイズ (size)                                                                 
    - **small**: 全身表示 - 導入・全体説明・非アクティブ時に使用                      
    - **medium**: 上半身表示 - 通常の解説に使用                                       
    - **large**: 胸上（近景）- 強調・感情表現・重要ポイントに使用                     
                                                                                      
    ### 感情 (emotion)                                                                
    - **excited**: 興奮・強調（自動的にlargeサイズになる）                            
    - **surprised**: 驚き（自動的にlargeサイズになる）                                
    - **happy**: 喜び・笑顔（mediumサイズ）                                           
    - **thinking**: 考え中（mediumサイズ）                                            
    - **neutral**: 通常（mediumサイズ）                                               
    - **concerned**: 心配・真剣（largeサイズ）                                        
                                                                                      
    ### ポーズ (pose)                                                                 
    - **normal**: 通常の立ち姿                                                        
                                                                                      
    ### 指定方法                                                                      
    - **セクション開始時**: small で導入                                              
    - **通常の説明**: medium + neutral/thinking                                       
    - **重要ポイント・強調**: large + excited/concerned                               
    - **質問への反応**: medium/large + surprised/happy                                
    - **長い発話**: parts で分割し、各partごとに演出を変える                          
                                                                                      
    ### parts分割の例                                                                 
    100文字以上の長い発話は、以下のように分割してください：                           
                                                                                      
    ```json                                                                           
    {                                                                                 
      "dialogue_id": "d003",                                                          
      "speaker_id": "sp1",                                                            
      "parts": [                                                                      
        {                                                                             
          "text": "導入部分の文章...",                                                
          "character_direction": {                                                    
            "size": "medium",                                                         
            "emotion": "neutral",                                                     
            "pose": "normal",                                                         
            "reason": "導入部分、通常の解説"                                          
          }                                                                           
        },                                                                            
        {                                                                             
          "text": "重要ポイントの文章...",                                            
          "character_direction": {                                                    
            "size": "large",                                                          
            "emotion": "excited",                                                     
            "pose": "normal",                                                         
            "reason": "重要ポイントを強調"                                            
          }                                                                           
        }                                                                             
      ],                                                                              
      "notes": "発話の意図",                                                          
      "emphasis_keywords": ["キーワード1", "キーワード2"],                            
      "slide_cue": "スライドの内容",                                                  
      "visual_cue": "画像提案",                                                       
      "pause_after": false,                                                           
      "timing_note": "話すスピードのヒント"                                           
    }                                                                                 
    ```

    **重要な注意事項**:                                                               
    1. **speaker_id** は必ず "sp1" または "sp2"                                       
    を使用（日本語の話者名は使わない）                                                
    2. **character_direction**                                                        
    は全てのdialogueに必須（partsがある場合は各partに）                               
    3. **100文字以上の発話** は必ずpartsに分割                                        
    4. **parts内では話者変更しない**（話者が変わる場合は新しいdialogueを作る）        
    5. JSON形式のみを出力し、他の説明文は含めない
