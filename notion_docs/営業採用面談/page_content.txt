二次面談対応フロー
  ※ 対象：営業（個別相談セールス / SnsClub提案）
※ 目的：1次面接の深掘り＋一緒に働くイメージ確認＋最終面接に向けた準備支援
※ 基本方針：見極め 5割 / アトラクト 5割
※ ゴール：「なぎさんに自信を持って送り出せる状態」にする
  0. 面談前に必ず確認（開始前30秒〜1分）
  1次面接の評価シートを確認
  特に深掘りしてほしいポイントを確認
  アトラクトで響いていたポイントを確認
  自分の中で「今日見たい論点」を1〜2個に絞る
  事前に見るべき項目
  ロジカル性（何点か / どこが気になったか）
  エモーショナル（熱量・想い）
  スキル面（営業経験 / Instagram知識 / 努力経験）
  条件面（オフライン / オンライン、アポ数、出社可否）
  1次担当者の申し送り事項
  1. 導入・1次面接の振り返り（3分）
  目的
  1次面接後の志望度変化を確認
  理解度と温度感を確認
  会話の入り方・印象を見る
  聞くこと
  1次面接を受けてみて、率直にどう感じましたか？
  1次面接を受けた後で、SnsClubの営業に対する印象に変化はありましたか？
  1次面接後に、気になったことや確認したいことはありましたか？
  見ること
  志望度が上がっているか
  会社 / 仕事理解が深まっているか
  受け身ではなく、自分なりに考えているか
  2. 改めて「なぜSnsClubの営業なのか」を深掘る（6分）
  目的
  動機の深さ・一貫性を確認
  “なんとなく応募”ではないかを見る
  最終面接に向けて、想いを言語化する準備を始める
  聞くこと
  改めて、なぜSnsClubの営業として働きたいと思いますか？
  その中でも、なぜ「個別相談」という仕事に興味を持っていますか？
  他の営業職ではなく、この仕事に惹かれる理由は何ですか？
  （SnsClub生なら）SnsClubのどんなところに魅力を感じていますか？
  見ること
  動機の深さ
  SnsClub / サービス理解
  愛着 / 共感 / 共鳴
  条件だけでなく、仕事そのものに惹かれているか
  3. 将来像・なりたい自分の深掘り（6分）
  目的
  会社の方向性との接続を見る
  3年後 / 5年後の視点があるか
  最終面接で話すべき“未来の言葉”を整理する
  聞くこと
  この仕事を通じて、3年後・5年後にどうなっていたいですか？
  その「なりたい自分」に対して、SnsClubの営業はどう繋がると思いますか？
  今のご自身に足りないものや、これから身につけたいことは何ですか？
  この仕事を通じて、どんな力を伸ばしたいですか？
  見ること
  将来像の具体性
  自己理解
  会社との方向性の一致
  目先の報酬だけでなく、中長期視点があるか
  4. 一緒に働くイメージ・営業観 / 価値観の確認（8分）
  目的
  「レベラーとして一緒に働けるか」を見る
  個別相談で何を大切にする人かを見る
  コミュニケーション・逆境への向き合い方を見る
  聞くこと
  個別相談で、目の前のお客様に向き合うとき、何を大切にしたいですか？
  あなたにとって「良い営業」って、どんな営業だと思いますか？
  これまで、うまくいかない時期が来たとき、どう乗り越えてきましたか？
  ご自身の強みと、逆にここは頑張らないといけないと思うところは何ですか？
  仕事をするうえで、周りからどんなタイプだと言われることが多いですか？
  見ること
  営業としての価値観
  押し売り型ではなく、相手に向き合えるか
  コミュニケーションの相性
  素直さ / 自己分析力
  逆境耐性 / 継続力
  一緒に働くイメージが湧くか
  5. 覚悟・コミットメントの確認（7分）
  目的
  最終面接前に“覚悟”を言語化させる
  本気度を確認する
  なぎさんに送る前の最終準備をする
  聞くこと
  この仕事をやると決めたら、何を覚悟する必要があると思いますか？
  この仕事で結果を出すために、自分が変えないといけないことは何だと思いますか？
  もし最初うまくいかない時期があっても、どう向き合いたいですか？
  最終面接で、なぎさんにどんな自分を見せたいですか？
  最終面接までに、自分の中で整理しておきたいことはありますか？
  見ること
  覚悟の深さ
  表面的な熱意ではないか
  自責で向き合えるか
  最終面接への準備状態
  “受かりたい”状態になっているか
  6. アトラクト＋最終面接への接続（5分）
  目的
  組織のビジョン / 方向性を伝える
  候補者の将来像と会社を接続する
  最終面接への期待感を高める
  伝えること（候補者に合わせて選ぶ）
  1次では伝えきれなかった、組織のビジョン・方向性
  なぜこの環境で成長できるのか
  教育体制 / ノウハウ / 伸びる理由
  候補者の「なりたい自分」が、このポジションでどう実現できるか
  最終面接は“見極められる場”だけではなく、“本気を伝える場”であること
  最後に確認すること
  最終面接に向けて、現時点で不安なことはありますか？
  最終面接までに、もう少し整理しておきたいことはありますか？
  今日話した中で、改めて「この仕事で実現したいこと」を一言で言うと何ですか？
  面談後に30秒で整理すること（超重要）
  1次面接で気になっていた点は解消したか：
  一緒に働くイメージが湧くか：
  候補者が覚悟を言語化できているか：
  最終面接でなぎさんに見てほしい論点：
  アトラクトで響いていたポイント：
  なぎさんに自信を持って送り出せるか（YES / NO）：
  最終面接に進める前の最終チェック（資料準拠）
  条件面（アポ数・出社頻度）をクリアしている
  ロジカル性・エモーショナル・スキル面に著しい懸念がない
  一緒に働くイメージが湧く
  候補者自身が覚悟を言語化できている
  「この人をなぎさんに自信を持って紹介できる」と言える
  時間が押したら、最低限ここだけ
  なぜSnsClub営業なのか
  3年後 / 5年後どうなりたいか
  個別相談で何を大切にしたいか
  この仕事で必要な覚悟
  最終面接でどんな自分を見せたいか
  二次面談トークスクリプト｜渡部広也さん
  ① アイスブレイク・温度感確認（3分）
  「今日は一次面談の内容も見ながら、もう少し深くお話できればと思っています。
  一次面談でかなり前向きに考えていただいている印象だったんですが、今の率直な温度感としてはどんな感じですか？
  他社さんも見ながらなのか、かなりLevelaに興味があるのか、その辺も含めて率直に教えてください。」
  ② 将来像の深掘り（10分）
  「一次面談で、
  ・35歳までに愛媛で年商100億の会社を作りたい
・今期1億を目指したい
  という話がありましたが、その辺りをもう少し聞かせてください。
  どんな会社を作りたいんですか？
  なぜ100億なんですか？」
  深掘り
  どんな事業をやりたいのか
  そのために必要な力は何か
  今の自分に足りないものは何か
  「営業ですか？
マーケですか？
採用ですか？
組織づくりですか？
マネジメントですか？」
  「今のご自身に一番足りないものって何だと思います？」
  ③ 自分でやることの限界と環境の価値（10分）
  「もちろん、自分で事業をやることもすごく大事だと思うんです。
  実際、渡部さんも行動しているし、それ自体は素晴らしいと思っています。
  ただ、正直、一人でできることって限界もあると思うんですよ。
  営業も、マーケも、採用も、組織づくりも、全部自己流で試行錯誤していくこともできます。
  でも一方で、
  実際に伸びている会社の中に入って、
  『どうやって事業が伸びるのか』
『どうやって人が増えていくのか』
『どうやって組織ができていくのか』
  を間近で見ながら学ぶ方が、結果的には成長速度って圧倒的に速いと思うんですよね。
  Levelaも、
  2期目から3期目で2億から15億まで約700％成長、
今期は40億を目指しています。
  正直、ここまで伸びている会社ってそんなに多くないと思っています。
  営業だけじゃなくて、
  人事、組織づくり、新規事業など、
  会社が伸びていく過程を近くで見られる環境ってすごく貴重なんですよ。
  私自身もそうなんですが、
  結局、自分だけで考えているより、
  伸びている環境に身を置いて学んだ方が、
  将来自分で事業をやる時の再現性って高くなると思っていて。
  渡部さんは将来的に大きな事業を作りたいという話でしたけど、
  今の段階で、
  『自分の事業を中心に自己流で進める』
  のと、
  『伸びている環境に深く入りながら成果の出し方を吸収する』
  のとでは、
  どちらの方が理想に近づけそうだと思いますか？」
  （本人に考えさせる）
  ④ 営業としての素直さと型化耐性（10分）
  「これまで自分の商品を売ってきた経験もあると思うんですが、
  逆にうまくいかなかった商談ってどんなものがありましたか？」
  「その原因って何だったと思います？」
  「もしLevelaに入った場合、
  最初は自分のやり方ではなく、
  まず型を徹底的に吸収してもらうことになります。
  自己流を一回横に置いて、
  素直に型を真似することってできますか？」
  「成果が出ない時期が1〜2ヶ月続いたとしても、
  ロープレやフィードバックを受けながら愚直にやり切れそうですか？」
  ⑤ 個別相談カウンセラーとしての価値観（10分）
  「渡部さんの『きっかけを与えたい』という話はすごく素敵だと思っています。
  個別相談って、
  単純に売る仕事ではなくて、
  その人の人生や悩みに向き合いながら、
  一歩踏み出すきっかけを作る仕事なんですよね。」
  深掘り
  「目の前の人に対して一番大切にしたいことって何ですか？」
  「売上と、相手の人生に向き合うことってどういうバランスで考えていますか？」
  「迷っている人に対して、どんな形で背中を押したいですか？」
  ⑥ コミットの話（10分）
  「さっきの話で、
  もし渡部さん自身が、
  『今は伸びている環境に深く入った方が将来に繋がる』
  と思うのであれば、
  自然と時間の使い方も変わってくると思うんですよ。」
  「今はご自身の事業が7割、Levelaが3割くらいという話もありましたが、
  仮にLevelaで本気で成果を出すとしたら、
  現実的にどういう時間配分になりそうですか？」
  「予定が重なった時、どちらを優先するイメージですか？」
  「半年〜1年というスパンで見た時に、
  Levelaから営業や事業づくりを吸収する期間にする、
  という考え方についてはどう感じますか？」
  ⑦ 最終面談に向けた整理・クロージング（5分）
  「私としては、
  若さもありますし、成長意欲もすごく感じています。
  一方で、
  今のまま自己流で事業を進めるより、
  伸びている環境の中で成果の出し方を吸収した方が、
  将来的な100億という目標にも近づけるんじゃないかなと思っています。
  だからこそ、
  『空いた時間でLevelaをやる』
  という感覚ではなく、
  『将来のためにLevelaで学び、成果を出す』
  という位置づけにできるかが大事だと思っています。
  もし次が最終面談になった場合は、
  凪さんからもおそらく、
  ・なぜLevelaなのか
・どれくらい本気でコミットできるのか
・自分の事業との優先順位をどう考えているのか
・Levelaで何を得たいのか
  この辺りは見られると思います。
  なので最終面談までに、
  整理しておいてほしいこと
  ① 自分にとってLevelaで働く意味
  ② Levelaを通じて何を得たいのか
  ③ 自分の事業との優先順位をどう考えるのか
  ④ どれくらい時間とコミットを寄せられるのか
  ⑤ Levelaで成果を出した先に、将来どう繋げていきたいのか
  この辺りは考えておいてもらえると、最終面談もより有意義になると思います。
  今日話した内容を踏まえて、
  渡部さん自身はどう感じましたか？」
