1. 結論
今回のサーベイで最も大きい論点は、**MGやサポート体制への不満ではなく、「講師業務の負荷が想定より重くなっていること」**です。
特に、以下が継続意向・おすすめ意向を下げている要因として見えます。
24時間返信・12時間アラートによる心理的負荷
Zoom / MTG / 講師トレーニングなどの拘束時間
生徒対応の範囲が曖昧で、どこまで対応すべきか分からない
本業・自分のアカウント運用・家庭との両立が難しい
オフ会や追加対応に対する報酬・交通費の納得感不足
営業時の説明と実態のギャップ
一方で、講師MG・サポートMGへの満足度は高い傾向です。
つまり、現場のMG個人の問題というより、
制度設計・業務設計・期待値調整の問題として見るべきです。
2. 主要数値サマリー
重要なのは、
成長実感・価値提供実感・MG満足度は高いのに、継続意向とおすすめ意向が伸び切っていない点です。
これは、
やりがいはある
  サポートもありがたい
  ただ、負荷が重く、他人には勧めづらい / 自分も続けるか悩む
という構造です。
3. 稼働負荷の分析
稼働負荷の回答
約4割が「重い」と回答しています。
さらに、稼働負荷別に見ると差が明確です。
「重い」と感じている層は、継続意向が大きく下がっています。
これは、退職・稼働縮小・紹介低下に直結するシグナルです。
4. 課題項目の集計
複数選択の課題では、圧倒的に「時間」が多いです。
この結果から、最優先で見るべきは、
サポート不足ではなく、時間負荷と業務範囲の不明確さです。
特に「時間」と「業務不明確」はセットで見た方がよいです。
単純に業務時間が長いだけではなく、
どこまで対応すればよいか分からないため、結果的に時間が膨らんでいる可能性があります。

6. リスク層の特徴
以下のいずれかに該当する人をリスク層として見るべきです。
総合満足度が2以下
継続意向が2以下
おすすめしたいかが2以下
リスク層に多い特徴は以下です。
稼働負荷が「重い」
課題に「時間」「業務不明確」が入っている
本業・家庭・自身のSNS活動との両立に不安がある
返信スピードやアラートにプレッシャーを感じている
報酬と稼働時間の見合いに疑問がある
ローンチや生徒対応への精神的負担が大きい
MGには満足しているが、業務設計自体に不安がある
特に危険なのは、
MG満足度は高いのに継続意向だけ低い層です。
この層は、人間関係の不満ではなく、
構造的な負荷で離脱する可能性があります。
7. MG別の見方
講師MG・サポートMGともに平均満足度は高いため、基本的には個人批判ではなく、
仕組み改善の材料として見るべきです。
ただし、以下の点は注意が必要です。
講師MG満足度が高い担当者でも、継続意向が低いケースがある
サポートMG満足度が高くても、おすすめ意向が低いケースがある
「該当なし」「未定」「まだ決まっていない」層では、評価が3になりやすい
担当がついた直後の講師は、良し悪しではなく「まだ分からない」で3をつけている
つまり、MGの品質改善だけでは限界があります。
サポート体制の有無・初期オンボーディング・業務設計の明確化まで見ないと、
継続意向は上がりにくいです。
8. 改善アクションプラン
優先度A：すぐ着手すべき
1. 返信ルールの再設計
現在の24時間返信・12時間アラートは、講師にとって心理的負担になっています。
見直し案
・通常返信：24時間以内を目安
・緊急対応：専用タグ / フォームで明確化
・土日 / 深夜：原則対象外、または返信義務ではなく確認推奨
・12時間アラート：講師本人のみ通知、または20時間時点の1回通知に変更
・講師MGへの通知：重大遅延のみ
・アラートは罰則ではなく、抜け漏れ防止のリマインドとして位置づける
ポイントは、
生徒満足度を落とさず、講師の常時監視感を下げることです。
2. 講師業務の範囲を明文化する
「どこまで対応するか」が曖昧なため、真面目な講師ほど疲弊しています。
講師が対応すること
・投稿 / リール / ストーリーの添削
・講義 / Zoomでの指導
・アカウント改善の方向性提示
・生徒の行動促進
・投稿改善に関する具体的なフィードバック
・クラス内での学習進捗確認
講師が抱えすぎないこと
・メンタルケアの深掘り
・家庭 / 本業 / 人間関係の相談対応
・契約 / 返金 / クレーム判断
・過度な個別リサーチ
・講師の専門外ジャンルの詳細戦略立案
・生徒の行動不足を講師だけで背負うこと
エスカレーション基準
・返金 / 解約 / クレーム：CSへ
・メンタル面の深い相談：コーチング / CSへ
・生徒の返信停滞：CSまたはサポートMGへ
・講師判断が難しい案件：講師MGへ
・業務範囲外の要望：運営判断へ
3. MTG / 講師トレーニングの棚卸し
すべてのMTGを一律で参加推奨 / 必須にすると、経験者・上位講師ほど負担になります。
見直し案
・新人講師：参加推奨 / 必須
・既存講師：任意参加＋アーカイブ確認可
・高実績講師：必要テーマのみ参加
・サポートMG相談済みの内容は、講師トレで重複相談しなくてよい
・固定MTGは、参加できない場合の代替手段を用意する
・月1回、全体MTGの必要性を棚卸しする
代替案
・アーカイブ視聴
・要点資料の共有
・別日質問会
・任意参加の相談会化
・経験年数 / 担当クラス数別の参加設計
優先度B：1ヶ月以内に整えるべき
4. オフ会・追加稼働の報酬ルールを整理する
オフ会は任意でも、担当生徒が参加する場合は実質的に負荷になります。
見直し案
・完全任意参加：報酬なし。ただし交通費補助を検討
・担当生徒の引率 / 対応あり：交通費支給
・運営から明確に依頼する場合：日当または時給支給
・登壇 / FB担当など役割あり：別途報酬
・任意参加と業務依頼の線引きを明確にする
「出す / 出さない」以上に、
なぜその扱いなのかを明文化することが重要です。
5. 営業時の期待値調整を見直す
講師側にしわ寄せが来ている可能性があります。
営業・CS側で明確にすべきこと
・生徒に求める最低稼働量
・講師が対応できる範囲
・返信頻度の現実的な目安
・講師は何でも屋ではないこと
・精神的に不安定な生徒 / 過度な依存傾向がある生徒の対応基準
・自走意思が低い生徒への対応方針
・クーリングオフ / 解約リスクが高い生徒の見極め
これをやらないと、講師側が
入会後の期待値ズレを吸収し続ける構造になります。
6. 新人講師のオンボーディングを強化する
0〜1ヶ月の講師は、「まだ分からないので3」という回答が一定数あります。
これは不満ではない一方で、
初期不安のサインでもあります。
新人講師向けに整えるべきもの
・初月の業務ロードマップ
・初回クラス運営チェックリスト
・よくある生徒対応テンプレ
・サポートMGへの相談例
・NG対応 / エスカレーション基準
・先輩講師の添削例
・講義例 / FB例の共有
・ローンチ前後のToDoリスト
優先度C：中期的に整えるべき
7. 講師タイプ別の稼働設計
全講師を一律ルールで運用すると、離脱リスクが高まります。
分け方の例
・本業インフルエンサー型
・副業会社員型
・主婦 / 子育て型
・フルコミット型
・高実績講師型
・新人講師型
設計すべきこと
・対応可能時間帯
・担当クラス数の上限
・MTG参加条件
・返信ルール
・サポートMGの関与度
・報酬設計
8. ハイリスク講師の個別フォロー

・継続意向が2以下
・おすすめ意向が2以下
・総合満足度が2以下

個別フォローで確認すべきこと
・何が一番負担になっているか
・継続するために何が変わればよいか
・担当クラス数を減らす必要があるか
・返信ルールの調整が必要か
・サポートMG / CSへの分業が必要か
・報酬や追加稼働への不満があるか

現状は、講師の善意・責任感・個人努力で支えられている部分があります。
ただ、今後さらに生徒数・講師数を増やすなら、以下の設計が必要です。
講師の役割定義
返信 / 稼働ルール
サポートMG・CS・コーチングとの分業
追加稼働の報酬基準

MTG / 講師トレーニングの再設計

10. 最終まとめ
今回のサーベイから見ると、現場のサポート体制自体は高く評価されています。
一方で、講師業務が拡張しすぎており、
「やりがいはあるが、負荷が重すぎて続けづらい」状態が一部で起きています。
したがって、次に打つべき施策は、MG教育よりも先に以下です。
・24時間返信 / アラートの見直し
・講師業務の範囲明文化
・MTG / Zoom / 講師トレーニングの整理
・オフ会 / 追加対応の報酬基準整備
・営業時の期待値調整
・新人講師の不安解消オンボーディング
・ハイリスク講師の個別フォロー
一言でまとめると、
講師満足度を上げるというより、「講師が長く続けられる業務設計」に変えるフェーズです。
クラス数ごとの稼働時間の状況
クラス数ごとの稼働時間を見ると、1クラスから一気に負荷が上がり、2クラス以上で20時間超が標準化している状況です。
クラス数別｜週の稼働時間
見立て
一番のポイントは、1クラスでも約4割が週20時間以上になっていることです。
本来、1クラスであれば10〜20時間程度に収まってほしいですが、実態としては20〜30時間、30時間以上も一定数います。
さらに、2クラス以上になると約7割が週20時間以上です。
つまり、2クラス以上を持つ講師は、かなりの確率で高稼働化しています。
稼働負荷の体感
少し意外なのは、2クラスより1クラスの方が「重い」割合が高い点です。
これは、クラス数だけではなく、以下の要因が負荷に影響している可能性が高いです。
新人講師で業務に慣れていない
初回ローンチ直後で対応が重い
生徒の自走度が低い
返信アラートや24時間返信の心理的負荷
Zoom、講師トレ、個別相談などクラス外業務が多い
自分の専門外ジャンルのリサーチが発生している
