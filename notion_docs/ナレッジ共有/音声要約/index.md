- 相談者（松村さん）が「収入を増やす副業」を探しており、案内役から“AI搭載アプリで競馬を資産運用にする”副収入サービスの説明を受けた。
- 「土日にアプリのスイッチをONにするだけ」でAIが自動で稼ぐと主張。
- 途中から友人が合流し、仕組みやリスクについて突っ込んで質問。
- 終盤、別の発言者（駒居氏）が登場し、相手企業の広告が自社を名指しで批判している件で抗議・責任者対応を要求。
# 提案内容（相手側の主張）
- 収益源：AIが競馬のレースでデータ分析→自動で利益を出す。
- 操作：毎週土日、朝9:30までにアプリを“ON”にするのみ。
- 入金：提携の銀行口座に週ごと入金。
- 会員数：3万人超（と主張）。
- 成果：最安プランでも平均月35万円、最上位で担当者は月90万円（と主張）。
- リスク説明：損失リスクは“ゼロではない”が、「これまで損失だけで終わった人はいない」と断言。
# 料金・プラン

- レベル1：49万9,000円（分割例：月約1万円）…月35万円程度が“安定”（主張）。
- レベル2：99万8,000円…月60〜70万円“期待値”（主張）。
