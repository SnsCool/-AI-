# ショート動画

[ ] 開始1秒に最もインパクトがある瞬間を持って来れているか？

[ ] 開始1秒に最後までみたいと思わせられる動線はあるか？

[ ] テロップはリアクションやキャプションに被っていないか？

[ ] 編集速度は1.5倍速以上になっているか？（ゆったりとしたコンテンツは離脱になる）

[ ] フォローまでの導線を動画内に敷いてるか？（他の投稿もみたいと思わせるorフォロー得点をつける）

[ ] 起承転結を意識できているか？

[ ] 音源はトレンドに沿っているか？

[ ] 他の発信者がバズらせている動画のネタ・構成は取り入れているか？

[ ] 他のSNSに転用をしているか？（TikTok・Youtubeショート）
