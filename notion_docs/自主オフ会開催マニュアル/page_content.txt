# 自主オフ会開催マニュアル

Notion URL: 
同期日時: 2026-01-21T19:18:34.490935

---

SnsClubでは生徒同士の学びや交流を深めるために、自主的なオフ会の開催を歓迎しています！

このマニュアルでは、企画から実施、記録までの一連の流れをまとめています！

こちらを参考に地方部屋、ジャンル別部屋、サークル部屋などで、是非積極的にオフ会を開催してリアルでの繋がりを発信に活かしてください✨

自主オフ会に関しましては運営は関与いたしませんので、幹事をしてくれる方に迷惑をかけないよう、また、トラブルが起きないよう、参加者一人一人が節度あるやりとりと責任のある行動をお願い申し上げます🙇🏻‍♀️

# 1.  Discordで参加を呼びかけ、日程調整をする

# 2.  LINEオープンチャットでグループを作成する

参加者をLINEオープンチャットに招待し、会場や時間などの詳細のやり取りを行なってください。

## 🔗LINEオープンチャット

# 3.  会場の予約を行う

会場を探すときはChatGPTを使うと便利です！

### ▼プロンプト（コピーしてご使用ください✨）

```html
以下の条件で、オフ会に使える会場を3つ提案してください。
カフェ・レンタルスペース・個室のある飲食店など、条件に合うものをお願いします。

・参加人数：（例：8〜10人程度）
・開催時間：（例：19:00〜21:00など）
・希望エリア：（駅名 or ざっくり「渋谷周辺」などでもOK）
・条件：（飲食OK・電源Wi-Fiあり・静か・おしゃれ・子供OKなど）
・予算感：（例：1人3000円以内、全体で1万円以内など）
・利用目的：（例：雑談中心、勉強会など）
```

# 4.  必要であれば名札を作成する

特に参加者が多い場合には、名札を用意しておくとスムーズに交流ができます！
SnsClubのオフ会で使用しているテンプレートもぜひご利用ください✨

#  5.  開催後、幹事の方は開催記録のスレッド作成を、参加者の方はアウトプットをする

当日撮影した写真をつけて、幹事の方はスレッドを作成して簡単なイベント内容を記載してください。

参加者の方はスレッドにコメントをする形で当日の学びのアウトプットや感想を送ってください。
