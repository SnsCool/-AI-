      - heading "オープニング" [ref=e22] [level=2]
      - paragraph: おはようございます、駒居です。本日は、質を追い求めても結果は出ないというテーマでお話をしていきます。
      - paragraph: 生徒さんから質問をいただきました。
      - paragraph: フィード投稿やリールを作成する際に、こだわりすぎてしまい、一つのコンテンツを作成するのに少なくとも数時間かかってしまいます。 毎日投稿を続けていく上でひとつひとつのコンテンツにそこまで時間が割けないことはわかっているのですが、どうしても納得いくまで作業を続けてしまいます。 駒居さん含めコンスタントに質の高い投稿を作られている方はどの程度まで妥協をして、どこまでこだわっていますか？よかったら教えてください。
      - paragraph: まずご質問いただきありがとうございます。
      - paragraph: 今まで、1年半ほどコンサルをしていく中で、この妥協点がわからなくてコンテンツに時間をかけすぎてしまう作成に時間をかけすぎてしまうというのは、本当に何度も聞いてきた質問ですが、皆さんには、質よりももっともっと圧倒的に大切なことがあるということに気づいてほしい、知ってほしいなと思います。
      - heading "質よりも大切なもの" [ref=e23] [level=2]
      - paragraph: コンテンツの質よりも大切なものって何なのかというと、それは量です。質を追い求めた結果、毎日投稿が続かない、質を追い求めてしまった結果、時間がすごくかかってしまうのは本末転倒なので、まずは質よりも、圧倒的に量を大切にしてほしいなと思います。
      - heading "なぜ質よりも量なのか？" [ref=e24] [level=3]
        - text: なぜ質よりも量の方が大切なのか？それは、
        - strong: 量をこなしていく中で初めて質が身につくから
        - text: です。
      - paragraph: 学生時代、部活動をしていた方が多いと思います。例えば、美術部に入られている方が「デッサンを書きましょう」って言われて一番最初からリアルな自画像のデッサンを、何年もやられてる方よりも質高く描けるかと言ったらまず無理だと思います。もちろん1回1回手を抜かずに書くということは前提ですが、ただ何枚も何枚も、何十枚、何百枚と描いていく中で、どんどんどんどん質が身についてきます。
      - paragraph: 私自身は学生時代バスケ部でしたが、バスケ初心者がいきなりバスケ上級者の人と同じようなプレイができるかというと、それも絶対に無理です。なぜ、バスケ上級者の人が上手にプレイできるかというと、超簡単で量をこなしたからです。
      - paragraph: これはコンテンツを作る上でも全く同じことが言えます。質問者さんは質を意識するあまり量ができていない、量ができなくなってしまうとおっしゃられていたので、これは本末転倒になってしまうかなと思います。
      - heading "量よりも大切なもの" [ref=e25] [level=2]
      - paragraph: 実は量よりもさらに大切なものがあります。それはスピードです。つまり、作業時間を短く仕上げることがとても重要になってきます。
      - heading "なぜ量よりもスピードなのか？" [ref=e26] [level=3]
      - paragraph: 量よりもスピードが大切である理由はたくさんありますが、一番大きな理由は「時間は皆さんの人生において、最も貴重な財産だから」
        - text: この前の大阪オフ会である生徒さんにアドバイスしたことがあります。それは
        - strong: 「2倍の時間を使って、コンテンツを作るのであれば、2倍以上の結果を出さなければならない」
        - text: という内容でした。要するに、コンテンツ制作に2倍の時間を使うのであれば、投稿を出した時に2倍以上のリーチ数、再生数を出さなければならないということです。2倍の時間かけてるのに1.5倍くらいの結果しか得られていないのであれば
        - strong: 「いつも通りの速さで2つのコンテンツを作った方が圧倒的に効率がいい
        - text: 」ということです。
      - paragraph: 量をこなす中で、作業スピードを上げていくかをぜひ考えていただきたいなと思います。
      - heading "妥協点について" [ref=e27] [level=2]
      - paragraph: 「質より量、量よりスピード」というお話をすると
      - paragraph: じゃあ、質はとりあえずどうでもいいのか！
        - text: と思ってしまう方がいますが、それはNGです。「どの程度まで妥協していいのか？どこまでこだわっていますか？」という質問の答えになりますが、
        - strong: 基本的に妥協は許されません！
      - paragraph: 「質を意識しすぎると結果は出ません！質よりも量が大切です！量よりもスピードが大切です！」と言っていたのに今度は「質を妥協するな」と…じゃあ私はどうしたらいいんだ…！
      - paragraph: という話ですが、焦らずにぜひ最後まで聞いてください！
      - heading "妥協が許されない理由" [ref=e28] [level=3]
      - paragraph: 「基本的に妥協は許されない」その理由は、皆さんのコンテンツはインフルエンサーと同等以上のレベルで作っていただかなきゃいけないからです。少なくとも、
      - paragraph: 自分なりに頑張ったんですけど…
        - text: というレベルでは現実は厳しく、一切リールは再生されないし、フィード投稿はエンゲージメントを取ることができません。では、どうすればいいのということですが、皆さんにはその解決策があります。それは
        - strong: 私たち講師に相談、フィードバック、添削依頼をすること
        - text: です。
      - paragraph: インスタ初心者の方はどれだけ質にこだわったとしても、僕たちプロが5秒で気づくミスに気づけなかったりします。例えば、
      - text: インスタ初心者が気づかない違和感
        - listitem:  テロップの位置がおかしい
        - listitem:  そもそも冒頭ワンシーン目の動画、これを使わない方が質が上がる
        - listitem:  音質が悪い
        - listitem:  画質が悪い
        - listitem:  光がなんか気持ち悪い
        - text: など、初心者の方がどれだけ質にこだわっても気づけないようなポイントは、
        - strong: 「僕たち講師に相談することで一瞬で解決できる！」
        - text: というケースがほとんどかなと思います。皆さんがある程度コンテンツ作成に慣れてきて、自分のコンテンツの良し悪しを自信を持って判断できるようになるまではこだわりすぎず、荒削りでもいいから、まずは量とスピードを意識して、私たち講師に添削依頼をしていただきたいなと思います。
      - paragraph: まとめると、質も大切ですが、皆さんがこだわるべきは量やスピード。そして、質を上げるためには私たち講師に相談をしてフィードバックをもらって、そのフィードバック通りに修正をすることによって質の担保していく。そういった流れが最も効率的に結果を出すアクションプランだと考えます。
      - heading "スピードを上げるためのアクションプラン" [ref=e29] [level=2]
      - paragraph: では、スピードを上げるためにはどうすればいいのか？具体的なアクションプランを8つご紹介したいと思います。
      - text: スピードを上げるためのアクションプラン8つ
        - listitem: フォーマットをテンプレ化する
        - listitem: 作業をテンプレ化する

        - listitem: 仲間と一緒に作業する

      - heading "その１「フォーマットをテンプレ化する」" [ref=e30] [level=3]
      - paragraph: まず１つ目は、「フォーマットをテンプレ化する」ということです。
      - paragraph: フォーマットのテンプレ化とは、フィード投稿の場合だったら「デザインのテンプレートを作りましょう」ということです。これは皆さん、授業の中でテンプレートを作って頂くので、特に問題はないかなと思います。
      - heading "その２「作業をテンプレ化する」" [ref=e31] [level=3]
      - paragraph: 2つ目は、「作業をテンプレ化する」ということです。例えば、これからリールを作りますという時には、アフレコをするタイミング、文字入れをするタイミング、動画を撮影するタイミング、こういったものが思いつきでバラバラのタイミングでやってしまうと、作業はどんどん遅くなってしまいます。
      - paragraph: 作業をテンプレ化することによって、より効率的に作業を進めることができます。例えば、私のリール作成作業テンプレは以下のようになっています。
      - text: 私のリール作成作業テンプレ
        - listitem: まずは台本を書く
        - listitem: 台本に合わせた動画撮影
        - listitem: キャップカットを開く
        - listitem: アフレコを入れる
        - listitem: 自動でテキスト文字起こし
        - listitem: 文字起こしで変なところがないかの確認
        - listitem: 動画を当てはめる
        - listitem: テロップのデザイン
        - listitem: アニメーションをつける
        - listitem: 細かいデザインの修正
      - paragraph: このように毎回、同じ流れで作業を行うことによって、効率的に進めることができます。
      - heading "その３「作業をまとめてやる」" [ref=e32] [level=3]
      - paragraph: 3つ目は、「一気にまとめてやる」ということです。
      - paragraph: 例えば、台本を書くだけの時間を用意してあげたり、動画編集は4〜5動画をまとめてやったり、その方がより作業スピードが上がる人もいるのではと思います。
      - heading "その４「作業を分業にする」" [ref=e33] [level=3]
      - paragraph: 4つ目が、「作業を分業にする」ということです。
      - paragraph: その３のまとめてやると同じようなポイントですが、分業とは1日のうちに、2時間かけて台本を書き、5〜6投稿分の撮影を一気に1日で行ってしまい、そこから動画編集・テロップ入れ・修正を5〜6投稿分一気にするようにして作業効率を上げている方もいるかなと思います。
      - heading "その５〜６「作業環境を整える」" [ref=e34] [level=3]
      - paragraph: 5つ目と6つ目は一気にいきますが、5つ目は「カフェに行く」そして6つ目は「誰かと一緒にやる」ということです。つまり「環境を変えよう」ということです。
      - paragraph: 例えば、私の場合は特に自宅で作業をしてしまうと、いつの間にかベッドで寝転んでしまってたり、いつのまにかYouTubeに夢中になってしまったり、いつのまにかゲームに手が伸びてるなんてことがあります。カフェに行ったり、作業友達と一緒にどこかに集まってやるなど工夫することで、自分が作業にマックスで集中できる環境を作るのも、スピードを上げる上で重要なアクションプランだと思います。
      - heading "その７「講師に添削依頼をする」" [ref=e35] [level=3]
      - paragraph: 7つ目は、「とにかくたくさんコンテンツを作って添削依頼をする」ということです。
      - paragraph: 先ほどもお伝えしましたが、自分の頭で考えて自分で解決しようとしても、思いつかないし、思いついたとしてもそれ自体が間違っている可能性もあります。自分で考えて考えて考え抜いて何時間もかけたコンテンツを、私たち講師に提出していただいて1からやり直しとなってしまうと心が折れちゃうと思うので、とにかく作って添削依頼をかけて頂くと効率が上がります。
      - heading "その８「他の生徒のフィードバックを見る」" [ref=e36] [level=3]
      - paragraph: 8つ目、こちらは一番できてない方多いと思いますが、「他の生徒のフィードバックも見る」ということです。
      - paragraph: 私自身、講師をやっていて感じていることですが、一つのクラスで何回も同じ質問を答えることがかなり多いです。「この質問、前に他の生徒さんにもフィードバックしたんだけどな〜」ということがあり、とてももったいないことだなと感じます。自分に対してのフィードバックではないとしても、他の生徒さんのフィードバックを見て聞き、自分ごととして取り入れると学びの効率がグーっとアップすると考えます。
      - paragraph: 各講師、確実に皆さんのためになるフィードバックをしていると思うので、それを見て自分も取り入れるということをやってあげると、同じ時間でも倍以上の学びがあると思うので、ぜひ他の生徒さんのフィードバックも自分の勉強になると思って目を配っていただけたらと思います。
      - heading "まとめ" [ref=e37] [level=2]
      - paragraph: 本日は「質を追い求めても結果は出ない」というお話をさせていただきました。
      - paragraph: 結論、質ももちろん大切ですが、特に初心者のうちは質を追い求めたとしても、そもそも質が上がらないので、量やスピードが大切だというお話をさせていただきました。SnsClubのコンサルサービス自体が皆さんの質をサポートするためのサービスになっているので、ぜひ、私たち講師を使い倒していただき、質も高いスピード感のあるコンテンツ作成に励んでいただけたらと思います。
      - paragraph: それでは今日も人生を変える1日にしていきましょう！
