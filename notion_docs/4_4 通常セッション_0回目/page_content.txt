よかったことや反省点、新たな気づき、改善しないといけないことなど、SnsClubでコミットするために何ができるか考えてみましょう!!
  コーチングセッションの議事録を、ご要望に合わせて整理いたしました。
  セッション議事録：藤本みや様（2026年4月4日）
  1. クライアントプロフィールと現状
  経歴: パティシエ歴30年以上のベテラン 。横浜プリンスホテル、グランドハイアット東京のオープニングスタッフを経て独立 。
  実績: グランドハイアット東京で3年連続最優秀社員 。世界大会での準優勝経験もあり 。著名なYouTuber（加藤純一氏、はじめしゃちょー氏、ヒカキン氏など）のウェディングケーキや制作物も担当している 。
  現在の活動: 新百合ヶ丘で店舗を運営し、今年8月で15周年を迎える 。
  最近の変化: パソコンを初めて購入し、息子の友人に教わりながら学習を開始 。また、技術向上のため今月末には高知へレッスンを受けに行く予定 。
  2. 今回の目標（ゴール設定）「現状の自分では到底達成できない、能力を引き出すためのゴール」として以下を設定しました 。
  ビジネスゴール: 会社の売上を現状の1.5倍にする 。
  SNS目標: Instagramのフォロワー数を13万人に到達させる 。
  波及効果: 売上向上により、従業員の給与や待遇を改善し、周囲の人々がより豊かな生活を送れるようにする 。
  3. SNS（Instagram）運用の方向性
  発信内容: 自身のハイクオリティな作品動画（リール）を中心に投稿 。これまでの実績（有名人との仕事など）を適切に紐付け、「もったいない」状況を解消する 。
  投稿頻度: 週3回を目標とする 。
  補足: 作品制作が難しい時期や、表に出せない仕事の期間は、オーナーシェフ（ご主人）のお菓子や、看板豚（店長）の投稿でアカウントの動きを止めない工夫をする 。
  4. 今後のアクションとルール
  コーチングセッション: 半年間で全13回（約2週間に1回ペース）実施。残り12回 。
  講師との1on1: 今回予約分を除き、残り5回実施可能（要マネージャー経由での調整） 。
  コミュニケーション: チャット（Discord）は原則24時間以内に編集・返信。忙しい際はスタンプでの反応も可 。
  学習の場: クラス添削会（2週間に1回）や、他者の視点を取り入れるための「グルコン」への積極的な参加を推奨 。
  5. コーチ（トミー）からのメッセージ「明日起きるのが楽しみになる人生」をモットーに、みやさんが決めた「13万人・売上1.5倍」という高い目標を全力で尊重し、一番の味方として伴走します 。みやさんの行動力とこれまでの実績があれば、必ず達成可能です。まずは「打席に立ち続ける（発信を止めない）」ことを大切に進めていきましょう 。
