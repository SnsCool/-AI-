## セクション一覧

Levela Portalは以下の6つのメインセクションで構成されています：

| セクション | 説明 |
|-----------|------|
| Levela全体 | 全体会議、メンバー紹介、MVV、図書館 |
| 部署 | CS部、マーケティング部、営業ポータル等 |
| 事業 | 各事業部門（AI教育、コーチング、運用代行等） |
| マニュアル | 業務マニュアル、ナレッジ共有 |
| インテリジェンス | 教材、AI活用事例、開発ツール |
| その他 | その他の情報 |
